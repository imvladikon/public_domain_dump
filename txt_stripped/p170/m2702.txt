


	
	מלאכים ממליכים מעריצים
	



מלאכים ממליכים מעריצים / משה אבן עזרא


מלאכים / ממליכים, / מעריצים / בכל פאות,



מקדישים / מרעישים, / מרגישים / ברב תשואות,



נוראים, / ואין נראים, / נפלאים / ונפלאות –



אומרים: קדוש קדוש קדוש יי צבאות!

שמו לעד / ברב-רעד / באימה הן / יעריצון,



שאון שרפים / ברום עפים, / כברקים / ירוצצון.



שואלים זה / רשות מזה / להקדיש אל במוראות,



אומרים: קדוש קדוש קדוש יי צבאות!

המון עדים / מרעידים, / ברגש י- / עריצו אל,



המלתם / קדשתם / בעת שלוש / קדשת אל,



בקול רעם / ברב נעם / בקול מראה / להתראות



אומרים: קדוש קדוש קדוש יי צבאות!
