מחברת עשרים וארבע

אמר המחבר: בהיותי בבית השר נטע נעמן / היה לי רע גואל ורחמן / רד עם אל ועם קדושים נאמן / ושמו צדקיה / נקבצו בו כל המעלות לחיה / וכמוהו לא היה לפניו / איש חמדות ובישן ועניו / נקי בכל ששה סדרים / ולו יד ושם בכל ארבעה ועשרים ספרים / חכם בדקדוק הלשון / ובסוד העבור היה הראשון / מה נוסיף לדבר עוד מאדותיו / והוא הפליא לעשות מכל אשר היו בדורותיו / ואהבני אהבה עזה / ואהבת כל איש זולת אהבתו היתה בעיני נמבזה / והיו לו שני בנים / בחיק היפי אמונים / היו נחמדים להשכיל ולעינים תאוה / וירשו מאביהם החכמה והענוה / היתה רוח אלהים בקרבם / ואביהם אהבת נפשו אהבם / ויהי היום ואני הייתי עם השר בחצר גנת הביתן / לראות הפרחה הגפן ואם הנרד ריחו נתן / ולפנינו סעדת לויתן / והנה איש עתי ובידו רצפה / במלקחים לקח לבי ומזיח שמחתי רפה / באמרו הנני שלוח אליך קשה / בבגד הזמן בבעל נפש קדושה / והרס לעברתו האבן הראשה / הנודעת לחסידות ולקדשה / קבר צדקיה החסיד והעניו / בחדש אחד שני בניו / ותחת היותו מעודו נותן אמרי שפר / ונלחם את קרית ספר / שם שק במתניו והוא יושב בתוך האפר / ועשתנותיו אבדות / ופניהם וכנפיהם פרודות / ולכן שמתי פני מועדות / להגיד אליך את כל התלאה אשר מצאתהו /

לא תדעי לך היפה / כי הארץ תחת ישביה חנפה / ויד האמות עלינו תקיפה / ואמת עשו בנו התאנפה / ומלאה כרסה מעדני לא נדע מה יהיה בסופה / היינו כחציר גגות ושדפה / ואין כירה שלא תהיה קטומה וגרופה / וכבר למדונו רבותינו כי מצות פדיון קודמת למצות עריפה / וכל שכן לאשר שדהו נסתחפה / וזרם מים כבירים שטפה / ורבותינו זכרונם לברכה הבדילו בין שן ורגל לנגיחה ונגיפה / ובין אוירא לתלות ולשרף אגופה / ובין ביתה לחצרה וקרפיפה / ובין אם קופא לגיו ובין לבר קופא / וחיבה התורה צאן ובקר בשחיטה ודגים סגיא להו באסיפה / ונאמר איש אל רעהו הבה נלבנה לבנים ונשרפה לשרפה / ונבנה עיר האהבה לעינים תאוה / ומגדל וראשו בשמי שמי האחוה / ונעשה לנו שם פן נפוץ ונהיה לראוה / אולי יש תקוה / ונתחכמה אל הפרוד פן ירבה / ובערה אש הקנאה ואין מכבה / ובמה תהיה אורה ושמחה וששון ליהודים / אם לא בהיות כל הארץ שפה אחת ודברים אחדים / ולכן מורינו ורבותינו הנכבדים / פנות קהל רומ"א העומדים על הפקודים / נשאי על סבלו / קחו איש מחתות שכלו / ותנו בהן אש האהבה והענוה / ושימו עליהם קטרת האחוה / והוליכו מהרה אל העדה / פן יצא קצף העדה המורדה / ותחל נגף הפרידה / ובזאת יכפר עון המורדים וזה כל פרי הסיר חטאתם באמרכם / שובו לאשר העמיקו סרה ובתתכם יד לאשר רחקו מעליכם / ובכן יביאו צוארם בעבודת הקהל והיה זה שלום / והאל עלת העלות / הנאדר בקדש ונורא תהלות / ולו נתכנו עלילות / הקדמון אשר לא סר מהיות / הבוחן לבות וכליות / ישמע קולנו מבור תחתיות / ויזכנו אל מי גלות עליות / שפע תושיות / כימי עולם וכשנים קדמוניות / ויחזיר עטרת קהלנו לישנה / וגדודי השמחות יבאו בארץ בא שנה / ופדויי אדני ישובון ובאו ציון ברנה / בחייכון וביומיכון יאמן / ובחיי דכל בית ישראל בעגלא ובזמן קריב ואמרו אמן /


כשמוע השר המליצה ההדורה אשר יפיה לא נחקר / היתה לו אורה ושמחה וששון ויקר / ויאמר זאת המליצה היא פרח כל המליצות אשר השמעתני / אמצא חן בעיניך כי נחמתני / בך ימצא איש כל מאוייו / הפורש ממך כפורש מחייו / ישבתי עם השר ימים ושנים והוא מתעלס באהבתי / ויהיו בעיניו כימים אחדים באהבתו אותי / אחרי כן גזר האל בפרידה / ולקחתי הנסיעה לי לצידה / והפקדתי השר ביד אל עילום / ואלך מאתו בשלום.
