


	
	מחברות עמנואל
	



מחברות עמנואל – המחברת העשרים ושבע / עמנואל הרומי


אמר המחבר: הייתי עם השר בימי הנעורים בעיר ויאנ”ה / ובהיותנו משוטטים ברחובותיה אחת הנה ואחת הנה / בחגים ובמועדים / ראינו חברת אנשים נועדים / חושבים שהם מדברים מדברי חכמה / ואין אתם יודעים עד מה / זה ישאל וזה יענה / זה יחרף וזה יגנה / זה יצעק וזה אליו יתאנה / קול מלחמה במחנה / אין ביניהם הסכמה מדבר / קצתם מספרים מעשים שהיו כבר / וקצתם אומרים מלך פלוני גבר / ובא בארצות ושטף ועבר / וקצתם אומרים אני הייתי בנהר כבר / וקצת נדיבי עמים נאספו / ולדבר חמדות נכספו / ויתנבאו ולא יספו / וקצתם לא בדעת ידברו ודבריהם לא בהשכל/ ויהי בלבי כאש עצור בעצמותי נלאיתי כלכל /



ואמר אל השר הראית מה הם עושים / כאלו הגרנות שוסים / או כגבורים במלחמה בוסים / אין בהם איש דבריו במשפט יכלכל / וכל אחד מהם עצת חברו יסכל / עד מה אנחנו מחשים / ולא נהיה אל המערכה נגשים / ונוציא מפינו התרשישים / הבה נרדה ונבלה שם שפתם / ונראה אליהם כי נבערה עצתם /ונדעה מה אחריתם / ונשמיעם פניני אמרותינו/ ונשחק בם כצפור ונקשרם לנערותינו /



ויאמר אלי השר אלהים יענה את שלומך / ועל טובך יזכר שמך / כמוני כמוך כעמי כעמך / אל כל אשר תטני / תקרא ואמר הנני /



ואמר אל השר ומה היא החכמה אשר בה


נדבר / וחילים עליה נגבר /


ויאמר השר אמצא חן בעיניך ונדבר מחכמת המזלות / ואז תראה אם אשיב הצל אחורנית עשר מעלות / ואם אדע שערי מכון ווילון / ואם אמר שמש בגבעון דום וירח בעמק אילון / כי אני הוא היושב בחכמת המזלות ראשונה / ויד כל העם באחרונה / ואל נא אשא פני איש ואל אדם לא אכנה / כי אני אהרס ולא יבנה / אשר אם יוכל איש למנות את עפר הארץ גם חכמתי כה תמנה / ולכן קרא ואנכי אענה / ואל חכמה אחרת לא נפנה / ואז תראה כי האנשים האלה לא ימצאו מענה / כי המקום מקום מקנה / וישמרו לפיהם מחסום/ כי לילה מחזון וחשכה להם מקסום /



ואמר אל השר משכני אחריך / הן לו יהי כדבריך /



ונקרב אל הזקנים והשרים / ואמר אליהם שמעו נא המורים / הקץ לדברי רוח/ או יהיה האדם או יהיה יברוח / היום הזה הוא לאדני קדש הלולים / לא עת לדבר דברי התלים / ולספר חלומות והבלים / או נדבר מדבר חכמה רמה או נחריש / הלא ידעתם כי מרדף ריקים ישבע ריש / הנה מה טוב ומה נעים כי מה שירשנו מן החכמה אותו נוריש /



ויחרישו העם ולא ענו אותי דבר / והזדון בלב קצתם גבר / וחלף בהם ושטף ועבר /



ויאמר השר תחת דבר היום דברי מהתלות / אחרי אשר חדלו הקולות / נדבר מן החכמות המהללות / ונזכור טבע שנים עשר מזלות / ותחת לשמע דברי האלה והאלון אשר בשלכת / ועתליהו המולכת / נדבר בטבע שבעת כוכבי לכת / ובסודות האל אשר גאלנו / ומבור שאון וטיט היון העלנו / אשר שמענו ונדעם ואבותינו ספרו לנו /



ואמר אל השר משכני אחריך / פתח פיך ויאירו דבריך /



ויאמר השר: מי כשבתי אל החכמה האלהיה / ומי כצדק מפתח שלחיה / ומי כמאדים על התנועה והנטיה / ומי כשמש על המלכות והעליה / ומי כנגה על היפי והרויה / ומי ככוכב על הכתב והמכתב ידו הויה / ומי כלבנה זבת חלב ודבש היא וזה פריה /



ואען ואמר: מי כשבתי בים דרכו / ומי כצדק יציץ מחרכו/ ומי כמאדים דם בשפכו / ומי כשמש על כוכבי אל במלכו / ומי כנגה בחשק לערך ערכו / ומי ככוכב שבט סופר בתומכו / ומי כירח חלק משמן חכו /



ויאמר השר: מי כשבתי מכה עמים בעברה / ומי כצדק ידריכם בדרך ישרה/ ומי כמאדים יוציא חרבו מתערה / ומי כשמש תופיע עלינו נהרה / ומי כנגה באש החשק כמו תנור בוערה / ומי ככוכב אצבע אלהים לחרות דברות עשרה/ ומי כלבנה תוציא ממזרים קרה /



ואען ואמר: מי כשבתי עוצר בעד הטפה / ומי כצדק צפה ראשה וחפה / ומי כמאדים מזיח אפיקה רפה / ומי כשמש רוח חיים בכנפיה שאפה / ומי כנגה צורתה רקם ויפה / ומי ככוכב ציר ראש וזנב אגמון וכפה / ומי כלבנה יוציא חתן מחפה /



ויאמר השר: מי כשבתי על החמורים / ומי כצדק על הגמלים והכרים / ומי כמאדים על העזים והשעירים / ומי כשמש על הסוסים האבירים / ומי כנגה על הפרדים הארורים / ומי ככוכב על הצאן והעדרים / ומי כלבנה על השורים/



ואען ואמר: מי כשבתי מעיד על שבתותי / ומי כצדק / לחזק כל בדק / ומי כמאדים / לשפך דם יקדים / ומי כחמה / לחדש פני אדמה / ומי כנגה / סביב לו נגה / ומי ככוכב / בין רגלי השמש כרע ושכב / ומי כירח / נטה ללון כאורח/



ויאמר השר: אין כשבתי עוקר מלכיות ומחריב / ואין כצדק צדיק וצדק יקריב / ולא כמאדים מחרחר ריב / ולא כשמש מזריח ומעריב / ואין כנגה בחשק ובזנוניו / ולא ככוכב קסת הסופר במתניו / ואין כלבנה דובר דבר דבור על אפניו /



ואען ואמר: אין כיום שבתי בימות השנה / ולא כצדק יהיה עם הצדק באמונה / ולא כמאדים מחזיק מגן וצנה / ולא כשמש עדינה / ואין כנגה זונה/ ולא צורה ברקיע כצורת כוכב קטנה / ולא מהיר כלבנה /



ויאמר השר: אין כשבתי משכיל ונהיר / ולא כצדק זהיר / ולא כמאדים זד יהיר / ולא כשמש זורח ומזהיר / ולא כנגה בשחקים מהיר / ולא ככוכב סופר מהיר / ולא כלבנה בין שורותים יזהיר /



ואען ואמר: אין כשבתי עצל ופסח / ולא כצדק נוצח / ולא כמאדים רוצח / ולא כשמש שואף וזורח / ולא כנגה חשק ושמח / ולא ככוכב סופר ופקח / ולא איש דברים ומליץ כירח /



ויאמר השר: שנות שבתי לקבורות / ושנות צדק צדק לרדף ולהורות / ושנות מאדים לנוע אחר הסחורות / ושנות חמה לרדף השררות / ושנות נגה לשגות בזרות / ושנות כוכב ללמוד התורות / ושנות הלבנה באש הילדות כמראה הלפידים בוערות /



ואען ואמר: השמש כחתן בתור המעלה / והלבנה תתכס בצעיף ותתעלף ככלה / וחמשת כוכבי אל כשושבינים משוררים בקול המלה / ושנים עשר נשיאים לעמתם יחולו במחולות וחוזרים חלילה / וצבא השמים מימינם ומשמאלם מריעים תרועה גדולה / והעיר שושן צהלה /



ויאמר השר: השמש כמלך והלבנה גבירתו / נגה פילגשו ואומנתו / וכוכב סופר מהיר במלאכתו / ושבתי יודע בינה לעתים / וצדק דן דין אמת לאמתו / ומאדים שר הטבחים והמזלות שרי צבאות מלכותו / וכל צבא השמים עמו ונחלתו / יושבים תחת גפנו ותחת תאנתו /



ואען ואמר: מי כטלה מוציא רוח מאוצרותיו / וכשור ישת חשך סתרו וסכתו סביבותיו / וכתאומים עשה לרוח משקל ומים תכן במדותיו / וכסרטן משקה הרים מעליותיו / וכאריה חזק וגבור בממשלותיו / וכבתולה ירדה העינה לשאב גשם מטרותיו / וכמאזנים ישקל הרי עולם וגבעותיו / וכעקרב בסופה ובשערה הליכותיו / וכקשת יורה יורה ומלקוש בקשתותיו / וכגדי תארכנה לילותיו / וכדלי קים ולא יחל דברו ומוצא שפתיו / וכדגים מאסף לכל מחנותיו/



ויאמר השר: טלה על ראשו משלו / ושור על צוארו על סבלו / ותאומים ידיו רב לו / וסרטן החזה גבולו / ואריה האסטומכא בגורלו / והבתולה הטבור


כלו / ולמאזנים הבטן וחילו / ולעקרב האשכים ומחי קבלו / ולקשת ירכתי אהלו / גדי הברכים ישבו בצלו / ודלי השוקים לחבלו / ודגים הרגלים יחנו באתות על דגלו /


ואען ואמר: יאמר טלה עליתי למעלה מכוכבי אל בעונתי / ולא עזרני אל זה קלותי /



ויאמר השר: יאמר שור עליתי אל השחקים בסבת על סבלי / ולא עזרני אל זה גבורה היתה לי /



ואען ואמר: יאמר תאומים הייתי בשחק בסבת האחוה / ובלעדיה הייתי לפני מלכים לראוה /



ויאמר השר: יאמר סרטן הייתי בשמים להיות לרשעים תבנית / להסב את לבם אחורנית /



ואען ואמר: יאמר אריה למדו ממני צניעות וגבורה כי אצלי בתולה ולא גערתי בה לעולם גערה /



ויאמר השר: תאמר בתולה אתי אהבת חסד והצנע לכת / כי לא פתוני שבעה כוכבי לכת /



ואען ואמר: יאמר מאזנים פלסו מעגל רגליכם בני / כי פלס ומאזנים משפט לאדני /



ויאמר השר: יאמר עקרב אם תהיו לרעות אחרי רבים / אני איסר אתכם בעקרבים /



ואען ואמר: יאמר קשת הושמתי בשמים לראות זיקים חצים ומות / לעוזבים ארחות ישר ללכת בדרך צלמות /



ויאמר השר: יאמר גדי ארבץ עם אריה וקשת ותתענג בדשן נפשי / לא ירעו ולא ישחיתו בכל הר קדשי /



ואען ואמר: יאמר דלי בלבבך עצה ולך דלי ואיך לא תדלה / ואני נשיאים ורוח גשם מקצה הארץ אעלה /



ויאמר השר: יאמר דגים לא העלני אל השמים רק השתיקה / ועתה בוא כתבה על לוח לבך ועל ספר חקה /



ואען ואמר: כמה גדולה מעלת הענוה / שהרי קדם טלה במזלות לכל בעלי העצם והגאוה /



ויאמר השר: מה גדלה מאד מעלת בעל היפה ואף אם לו קרנים / שהרי הושם שור מקרין מפריס מזל בשמים /



ואען ואמר: מה גדל חשק החשקים הנתאמים / שהרי הושם בשמים מזל תאומים /



ויאמר השר: אין לתמה אם נדיבי עמים ילכו קדורנית / אחרי היות סרטן בשמים וחוזר אחורנית /



ואען ואמר: מה גדלה מעלת האיש אל תכלית הגבורה יגיע / שהרי הושם אריה מזל ברקיע /



ויאמר השר: מה גדלה מעלת הבתולה / שהרי בין מזלות השמים תחנה באותות על דגלה /



ואען ואמר: איך לא יראו ולא ייראו הרשעים והם בעלי העינים / כי פלס ומאזנים לאדני בשמים /



ויאמר השר: אין לתמה אם יעלה לשמים שיא הרשעים / אחרי היות עקרב למעלה משבעה רקיעים /



ואען ואמר: אין לתמה על הגזרות החלות ביבשת / אחרי היות בשמים המורים בקשת /



ויאמר השר: אין לתמה על בשר הגדי אם גדלה מעלתו / אשר למעלה מכוכבי אל תחנותו /



ואען ואמר: לא ידעתי איך העבים ימלאו גשם ויריקו מים / עד ראיתי מזל דלי בשמים /



ויאמר השר: לא ידעה נפשי איך עלו הדגים בשמים / עד ראיתי כוכבים בדלי יוצאות לשאב מים /



ואען ואמר: איך איש לא ירחף על גוזליו וירחם / בראותו שמים עיש על בניה תנחם /



ויאמר השר: ראיתי בשחק דלי ואין בור ואין מים / ואולי כי הוא לדלות ולהעלות נפש הצדיק בשמים /



ואען ואמר: נפלאתי בראותי כי אין צורת עוף במזלות ולא ידעתי לדבר פשר/ עד אשר הבחנתי כי זה מיראת כוכב הנשר /



ויאמר השר: לא נברא מאזנים ברקיע / רק להיותו מרך בלב הרשעים מגיע/



ואען ואמר: אתמה אחרי היות שבתי לברואים משטח חרמים / איך עלה בגורלו מבחר הימים /



ויאמר השר: לא תמהתי איך הזהרו מעשות מלאכה בשבת אבותי / אחרי אשר הבחנתי בשכלי עצלות שבתי /



ואען ואמר: אין לתמה אם הימים פרשו רשת ליד מעגל / אחרי היות שבתי ומאדים בגלגל /



ויאמר השר: פלוני יתחבר פעם בחדש עם העדינה / כהתחברות שמש פעם בחדש עם הלבנה /



ואען ואמר: פלוני צנוע כשמש החונה בכל שנה חדש בבתולה / ולא שת לבו לעשות עמה נבלה /



ויאמר השר: יאמרו בקדרות השמש דברי חלומות והבלים / ולא ידעו כי פני הצביות אורו גוזלים /



ואען ואמר: לא עלה בלבי לבזות ולשקץ איש קצר קומה וענותו / מן היום אשר ידעתי ערך גוף כוכב חמה ומעלתו /



ויאמר השר: ראיתי פעם ופעמים אור הלבנה גנב / ולא ידעתי הסבה כי תשקיף מן האשנב /



ואען ואמר: לכוכבי גלגל השמיני נפש צנועה / אשר עמדה בתולה ביניהם חמשת אלפים ושמנה ושמנים שנה ואיש לא ידעה /



ויאמר השר: אחשב היות כוכבי השמים זקנים או סריסים / אם שכנה בתולה עמהם ולא עשו לה כתנת פסים /



ואען ואמר: אין לתמה אם יש בארץ כסילים למיניהם / אחרי ראותנו כוכבי שמים וכסיליהם /



ויאמר השר: אין לתמה אם מעטו גדולי המעלה ורבו השובבים / יש ברקיע שמש אחד ושני כסילים ושני כלבים /



ואען ואמר: אחשב כי הכלב בשמים יהיה / לשמר טלה וגדי מפי אריה /



ויאמר השר: אחשב כי לא נברא אריה בשחקים / רק לשמר בתולה מן החשקים /



ואען ואמר: לא היה אריה אצל בתולה / רק מדאגה מדבר פן תעשה נבלה/



ויאמר השר: אלו איש על הנשים נאמן יהיה / היה אצל בתולה תאומים ולא אריה /



ואען ואמר: מה גדלה מעלת דגים על כל המזונות / שהרי הושם במזלות מאסף לכל המחנות /



ויאמר השר: תמה אני איך לא ישמרו האמות שבתותי / בראותם כי העליון מכוכבי לכת הוא כוכב שבתי /



ואען ואמר: תמה אני איך יהיה דרך הרשע כמשכת חדק / בראותי בשמי השמים כוכב צדק /



ויאמר השר: אין לתמה אם הצדיקים את הרשעים מנדים / שהרי הושם בשמים צדק על מאדים /



ואען ואמר: לא יתגאה המלך לחן בני תמותה לעתות / כי הנה השמש נצוצי חשמליו חבקו אשפתות /



ויאמר השר: איך לא יהיה לחשק אחרי נמכר גאלה ופדיום / ואור נגה הולך ואור עד נכון היום /



ואען ואמר: לא יבוש שפל רוח להתחבר עם בעל מעלה רמה / כי הנה כוכב הקטן שבכוכבים לא יפרד מן החמה /



ויאמר השר: אין לתמה אם היעלה הלבנה תהיה עם החשקים באמונה / כי הנה לא יתחבר השמש פעם בחדש רק עם הלבנה /



ואען ואמר: אין לתמה אם העקרב לשפך דם יקדים / אחרי היות עקרב בית מאדים /



ויאמר השר: נקראו בעלי קרנים אנשי בוגדות אשר לא עמדה רגלם במישור / מפני היות בית נגה המוכרת גוים בזנוניה שור /



ואען ואמר: אין לתמה על מאדים אם ידיו מלמדות לקרב / אחרי היות בביתו עקרב /



ויאמר השר: אין לתמה אם היפה כלבנה תחזר אחורנית לעת זקנה ותקטן / אחרי היות בית לבנה סרטן /



ואען ואמר: השאלים מדוע החשקים זלים זהב מכיס ולא יחמלו / כי בית נגה מאזנים אשר כסף בקנה ישקלו /



ויאמר השר: אין לתמה אם הדג הקיא את יונה על היבשה / כי אחרי היות דגים בית צדק היה להם הצדק למורשה /



ואען ואמר: אין לתמה מן הירח אם הוא לפעמים במהלכו נזור / שהרי ביתו סרטן והוא אחורנית נזור /



ויאמר השר: אין לתמה אם השמש כמלך על תבל יהיה / אחרי היות ביתו אריה /



ואען ואמר: אין לתמה על האריה אם הוא מלך על חיות האדמה / אחרי היות אריה בית חמה /



ויאמר השר: אין לתמה אם הבתולה תתיפה ככלה / אחרי היות בית כוכב חמה בתולה /



ואען ואמר: תדמה השמש לחתן יכהן פאר בגלגליה / והלבנה לכלה תעדה כליה /



ויאמר השר: אראה חמשת כוכבי הנבוכה / כשושבינים העומדים על הברכה/



ואען ואמר: אראה שתים עשרה צורות אשר באפדת גלגל המזלות / לדמיון שרים וחוללים יחולו במחולות /



ויאמר השר: אראה צבא השמים מימינם ומשמאלם כחול / לדמיון העם הנקבצים לראות המחול /



ויהי כשמע העם את מליצותינו החמדות / והחכמות הרמות והנכבדות / עמדו מרעידים ותמהים / כאלו שם נגלו עליהם האלהים / ויעמד אחד מראשיהם במצות אחיהם לברכנו / ולהודות אל מעלתנו / ויאמר שלום לכם שלום הררי אל / ראשי אלפי ישראל / ברוך אשר חלק לכם מכבודו / ויאצל עליכם מהודו/ לא ידענו בלתי היום כי שוכני בתי חמר אשר כעפר יסודם / ידעו מהלכות צבאות השמים וסודם / חשבנו


כי השמים שמים לאדני והארץ נתן לבני אדם / אמנם היום הזה ראינו נפלאות / ושמענו מכוכבי שחק נוראות / לא נשאל על מעלתכם מופת או אות / אין לתמה אם היינו בעיניכם לחרפה ולקלס / שקלתם גבעות עולם במאזנים והררי עד בפלס / לשמע אזן שמענו שמעכם ועתה עינינו ראתה / מי לא יירא ממעלתכם כי לכם יאתה / דברתם היום נגידים / הננו לאדנינו לעבדים / ועלינו ועל פעל ידינו תצוו / כי לכם הלבבות יקוו / ומדשן חכמתכם ירוו /


אחרי כן קם העם והשתחוו / ואנחנו הפקדנום לאלהים /


והם נשארו על מעלתנו תמהים / ויהי כי באנו אל המלון ויאמר השר הראית איך חשפתי היום סודי / ואיך בחכמות המזלות גברה ידי / מי יתן ותגבר ידי בשירות המהללות / כאשר גברה בחכמת המזלות / אמנם אם איש מהם לדבר בענין השיר יאבה / אצטרך לבא חדר בחדר להחבא /


ואען ואמר חייך השר אם איש מהם יבא ומענין השיר ישאל / אשים מלתו לאל / ואת גאלתך אנכי אגאל /



עודנו בדברים האלה מדברים / והנה עשרים אנשים כאיש אחד חברים / באו עדינו / לחשף סודנו / ויאמרו אלינו אדונינו / היום ראינו מכם נוראות / אשר לא ישבע איש לדבר ולא תשבע עין לראות / ועתה ייטב בעיניכם להודיענו על הנכונה / אם במלאכת השיר ידיכם אמונה /



ויאמר השר יש ויש / הנה חברי לא יתביש / למדתיו מן השירים כבר / מה שידע להשיב שאלו דבר / ועתה יחשף כל אחד מכם סודו / ואז תראו כי רב חילו וכי כביר מצאה ידו /



ויגש הראשון ויאמר אדני אני מעיר אחת ושמה ביתתר / ובשכונתי עלמה ושמה אסתר / דיה לכל המלאכה לעשות אותה והותר / מצאה חן בעיני מכל הבתולות / קומתה דמתה לתמר ושדיה לאשכולות / היא יעלת חן ויעלת אהבים / מעשה חשב עשה אותה כרובים / ושדיה כדתן ואבירם יצאו נצבים / קרעו מעליהם את עדים / לא ידעתי אם הם אבלים לעורר בכים / או להראות העמים והשרים את יפים / כי ליפיה בתבל פרצה פרץ / אין כמוה בכל הארץ/ ולו תאמר שמש בגבעון דום וירח בעמק אילון / לא תמצא נגע וקלון / וכל אחד מהם מאמר אסתר קים / וכמוה לא נראית מים עד ים / ועתה הואל וחבר עליה שיר נפלא / אשר בכתם אופיר לא יסלא /



ואקום ואחבר עליו זאת השירה / וישבע כי לא יתן סגור תחתיה ולא ישקל כסף מחירה /

  

ואשא משלי ואמר:

  

שאלוני על מה שדי זאת העלמה



לבושם קרעו לעורר את בכים



האם קודרים על אם יהו כאשר נראם



קרועי בגדיהם לנצל את עדים



עניתים קרעו וראשם פרעו



להראות העמים והשרים יפים



ואמרו מה טבם עניתים מה טובם



דבש וחלב זבות והנה לך פרים



הלא זאת היא אסתר לחשקים תעתר



וכמה לא תחזה מים עד ים



ואלו דום שמש ואיש חשך ימש



תצו היפה מאמר אסתר קים .

  

ויגש השני / ויאמר אדני / יש לי שכן ושמו שלמה / ומדי דברי בו ואזכר מומו / יחרד לבבי ויתר ממקומו / ולו היה בדור איש אשר רשעי הארץ יצמית / שכני היתה אחת דתו להמית / ושם אשתו שלומית / ושניהם מאנו לשוב החזיקו בתרמית / כי האיש ההוא לחרופים ולגדופים / והאשה כלה נאופים / בהם כל דפי וכל גנאי / רעים וחטאים לאדני / וחפצתי לחבר שיר אחשף סודם / להגיד חסדם ונפלאותם לבני אדם / ודרך השיר אלי צר אין דרך לנטות ימין ושמאל / וידעתי כי אין זולתך לגאל / ולכן חבר עליהם שיר כאשר עם לבבך / ואל שדי יברך אותך ויפרך וירבך /



אז נפשו מדשן שירי רויתי / והנבאתי כאשר צויתי /

  

ואשא משלי ואמר:

  

בעת תחפץ ראות שועל מרמה



ואשת רע תיחד לנבלה



לכה אל בית שלמה איש שלומית



ושם תראה חצר שועל ובלה .

  

ויגש השלישי ויאמר אדני יש-לי שכן אחד גבן שובב / רום עינים ורחב לבב / כאלו היה מבני חבר הקיני מבני חובב / יגאה כשחל ושכח מומו / בעבור היות המשרה על שכמו / חשב כי זה האביון לבשתו / כי תשב באיתן קשתו / וכתפו אשר משכמה תפל ודבשתו / יהיה ההר חמד אלהים לשבתו / וחפצתי לחבר לו שיר אגלה בו מומו / ומאז לא אזכרנו ולא אדבר עוד בשמו / וידעתי כי כל תוכל / כי שולי שירותיך מלאים את ההיכל / וכל הנחרים בך תחת רגליך שתה / וכל לשון תקום אתך למשפט חתה / ועתה אל תנקהו כי איש חכם אתה /



ואקום ואחבר-לו זה השיר / ויאמר ברוך אדני ואעשיר /

  

ואשא משלי ואמר:

  

משרה עלי שכמך נגידי היתה



מאז ומה יועיל לך ברחך



ברוך אלהים חי בחמלתו לך



נתן שכם אחד עלי אחיך.

  

ויגש הרביעי ויאמר אדני יש-לי אשה יפה / כמו שחר נשקפה / פוריה וענפה / פריה למאכל ועליה לתרופה / אהבתיה ואהבנה / ומיראתי פן תבגד בי לילה ויום אצרנה / וקרוב אל ביתי איש ושמו יצחק הבבלי / והוא ישב ממולי / וחלק משמן חכו / ומדי יום ויום יציץ מחרכו / ויביט כה וכה / יארב במסתר כאריה בסכה / ומול אשתי עפעפיו יישיר / פעם ינגן ופעם ישיר / והיום אשר פעמים ושלש לא יראנה / יחשב כי ממרום שלח אש בעצמותיו וירדנה /


וחפצתי לחבר שיר בו אודיענו / כי כבד עלי המעשה אשר יעשנו / ואם עלה עצם בגורלי יגררנו / ויניח לי הצביה אשר עלתה בגורלי / כי היא לבדה חלקי מכל עמלי / ולכן אדני אם תחפץ באהבתי / גאל אתה את גאלתי / ותשוב תנחמני ותרב גדלתי /


ואקום ואחבר לו זה השיר בלשון ארמי / ויקחהו וילך מעמי /

  

ואשא משלי ואמר:

  

מה ליה ליצחק רחמתי למחזי



הא לא חמיתי לעלם דביתהו



יחדי בחולק דסליק לעדויה



אנן בדידן ואינהו בדידהו .

  

ויגש החמישי ויאמר אדני היו לי שני חשוקים / במעבה החן יצוקים / וכראי מוצק חזקים / ומנפת צוף מתוקים / מה נעמו ומה נאוו / בלכתם קוממיות ולא כרעו לי ולא השתחוו / שגיתי באהבתם ובכל עת אותי רוו / וכאשר הורידו לארץ ראשם ולי השתחוו / מאסתי חברתם / ומאסתים וגם געלתים לכלותם / להפר בריתי אתם / ורציתי שתחבר על זה שיר נמרץ / אשר כל אזן לשמעו תחרץ /



ואען ואמר חייך החידה אשר תאמרנה / הגיעה ליד איש יפתרנה /

  

ואשא משלי ואמר:

  

שדי עפרה בלכתם קוממיות



פדות לבי ונפשי בחלבם



ומן היום אשר השתחוו לי



מאסתימו ונפשי בחלה בם .

  

ויגש הששי ויאמר אדני יש לי רע משכיל וזרע קדושים / והוא החרוץ והפקח שבאנשים / ושנותיו הגיעו לחמשים / העביד שכלו לתאותו / וירעשו הרים מגערתו / ומנעוריו רדף אחרי ההבל / ויאמר טוב ללכת אל בית משתה מלכת אל בית אבל / והיה כנור ונבל / תף וחליל ויין משתיו / כלה בתהו ובבהלה שנותיו / והוא נחש ינחש / אלה וכחש / ורצח וגנב ונאף / שמות ושאף / ותמהתי ואתמה איך מן המות לא יחרד / בראותו כי יומו מאד רד / ובקרוב יהיה מן העולם הזה נפרד / וחפצתי לשלח לו שיר ולהגיד לו / עד מתי ינבל פאר שכלו / ושכח כי בפתע ינתק מבטחו מאהלו / ותשכן באהלו מבלי לו / ומאומה לא ישא בעמלו / ועתה אדני ירם הודך / הואל ופנה אלי בגדל חסדך/ וחבר על הענין המתאר


/ שיר מפאר / כי ידעתי את אשר תברך מברך ואשר תאר יואר /


ואקום ואחבר אליו זאת השירה / ויענדנה לראשו עטרה / וישלחה לרעהו אשר לתתו לו בשורה /

  

ואשא משלי ואמר:

  

לעד מה תנבל פאר שכלך דוד



ואיך לבך לא ירופף וירעוד



אשר תחרץ תהמה על דבר מום



ואלה וכחש ונאף וסעוד



ואת בן חמשים וזרע קדושים



והחל אשורך להמוט ולמעוד



נטה מעשות רע צבי חן ושוב מ-



צבא העבודה ולא תעבד עוד .

  

ויגש השביעי ויאמר אדני יש-לי שכן שותה שכור / יתן הילד בזונה והילדה ביין ימכר / חפץ שיטפו ההרים עסיס / ומשנאתו המים ישנא הטל והרסיס / ואין לו בדבר מדברי החכמה טכסיס / ונשי ביתו הם לרואיהם צער / תדמינה לחיות היער / רעמו פנים סערו סער / בביתו הרגיעה לילית וקננה קפוד / ויש עת לבכות ועת לספוד / שם נקבצו דיות / ולולי שחסר האופנים היו לו ארבע חיות / ועתה חבר אלי שיר אשלחנו אליו / לתת לו כדרכיו וכמעלליו /



ואקום ואחבר לו שיר זה מופז ערכו / ויפן וילך לדרכו /

  

ואשא משלי ואמר:

  

בית הצבי שם נקבצו דיות



ליות ושמה קננה קפוז



לו יטפו הרים עסיס ישתם



הן יעשק נהר ולא יחפוז .

  

ויגש השמיני ויאמר אדני יש לי רע ועמית / אחת דתו להמית / ואתמה איך הארץ אותו תסבל / ואיך כנבל עלה מגפן לא יבל / כי באחיו חנם יחבל / ויעבר תורות ויחלף חוק / והיו בעיניו לשחוק / ושכח כי מרשעים אדני רחוק / ואם עד כה העלים האל ממנו את עיניו / כאלו לא ראה ערות זנוניו / אין שכחה לפני כסא כבודו / ופתאם יקום אידו / ויראו עיניו כידו / ואיך שכח בן הזונה / כי מרה תהיה באחרונה / כי לאדני השמים ושמי השמים / ועבר על פשעו פעם ופעמים / ואם לא ישוב חרבו ילטש ועל הארץ ישפכנו כמים / וחפצתי אשר תחבר בו שיר אעוררהו / אולי מדרכו הרעה אסירהו /



ואמר אליו הנה השיר אשר תבקש / להסיר מעמיתך לב עקש /

  

ואשא משלי ואמר:

  

אם בעבר תורות וחק לחלף



צור העלים עין וחטא נשא



שוב מזדון לבך ארי לאו כל



שעתא ושעתא מתרחיש נסא.

  

ויגש התשיעי ויאמר אדני חפצתי שתחבר אלי תפלה קצרה / אתפלל אותה בעת צרה / אתקרב בה אל האלהים / המרים שפלים ומשפיל גבהים / תהיה ממבחר המליצות חצובה / ומזמרת הענינים קצובה / ואם תשא משאת התפלה הזאת אלי מאת פניך / ידעתי כי מצאתי חן בעיניך /



ואען ואמר הנה השיר כחפצך / ואדני אלהיך ירצך /

  

ואשא משלי ואמר:

  

ממי אלהי אשאלה שאול



חפצי ואין זולתך לגאול



לו חסדך לא תחיה רוחי



בדי עצמי ירדו לשאול



לולי אספתני ברב חסדך



מתי בבין צרעה ואשתאול



צלך לבד אשאף ולימין



אסור בסור הבוערים לשמאל



ארחץ בבור כפי ביום ערלי



לב בזמן יתגאלו גאול



אם כופרים רשו ורעבו



נא עבדך אל תעזב לשאול .

  

ויגש העשירי ויאמר אדני ידעתי גם ידעתי כי הנפש יפעתה ברה וטהורה / עטרת צבי צפירת תפארה / ערוכה בכל ושמורה / מצור השכל חצבה / ומלהבות חשמליו נקצבה / ובמרום המעלות נצבה / אצל אותה האל מהודו העליון / ונגה כאור תהיה וקרנים מידו לה ושם חביון / ובחמלתו עליה עשה לה בעולם השכל אפריון / להשיג המרגשות באמצעות הדמיון / וכתב לה גליון / וכאשר ישלחנה בזה העולם האביון / השפל ואפל ואין נגה לו רק קצף וכדי בזיון / תשמח בו כשמחת יונה על הקיקיון / ותשכח העניה והדלה / המקום אשר היה שם אהלה בתחלה / ותחשב כי באה אל המנוחה ואל הנחלה / וכאשר יעורר השכל הנפשות האנושות לאמר / הלא תזכרו הר המור / בית אלהים אשר שם היינו / תתנכרנה ותאמרנה העוד לנו חלק ונחלה בבית אבינו / ולכן באתי אליך השר הנעלה / לבקש אחר בקשת המחילה / לחבר שירה יהיה זה ענינה / אשפיל בה הנפש הנענה / ואזכירנה את עונה / וירד הדרה והמונה ושאונה / ואם תחברנה כפי שכלך / הנפש והרכוש קח לך /



ואען ואמר לא חפצתי ממך כסף ורכוש / ואקום ואחבר לו שיר לא יערכנו יגיע מצרים וסחר כוש /



וישמח עליו האיש ואליו ערב / כמוצא שלל רב /

  

ואשא משלי ואמר:

  

נפשות בני אדם אשר לא בנו



סוד בית אלהים כאשר היינו



מתנכרות אומרות העוד יש לנו



חלק ואם נחלה בבית אבינו .

  

ויגש האחד עשר ויאמר יש אילת אהבים ויעלת חן בשכונתי / הייתי נותן באהבתה שדה מקנתי / כי לה שתי עינים / קורעות הלבבות לשנים / וחליתי באהבתה והחדלתי את דשני / כי בראותי בשפתותיה את תקות חוט השני / ואראה ברד השנים אמות לשחת / כי אראה ברד ואש מתלקחת / ולכן אחרי אשר אני בים חשקה נטבע / הנני בגאוני נשבע / כי את קובעי נפש אקבע / ולו תהיה אשת איש הבינים / אצמיח לו קרנים / ולכן חושה השר והיית לי לעינים / וחבר שיר נחמד לרואיו / תכספנה הלבבות לשמע פלאיו /



כשמעי נתן החשק בפי חזון נפרץ / ואחבר שיר נמרץ / ואז האיש לקחתו חרץ / ויאמר אלי לו תחפץ ככר כסף אשקל / ואען ואמר חנני אלהים ויש-לי כל /

  

ואשא משלי ואמר:

  

אל יעלת החן שתי עינים



הקורעים לב חשקים לשנים



ברד ואש התלקחו בה הם שפת



שני אשר לה על ברד שנים



אחר אשר נפשי לחשקה תהמה



צרה כמבכירה עלי אבנים



אעש ולו אשת רפאים תהיה



על הרפאים עשתרות קרנים .

  

ויגש השנים עשר ויאמר בי אדני צר לי מאד ולנפשי ריבותי / ואשר קרה לי מן הרע אנכי סבותי / בידעי מעלת הנפש ולא בינותי / והיות הגוף טיט ורפש ומסבל שכמו הסירותי / מכרתי הקים בכלה והטוב ברע המירותי / ומרחם כרשעים זרתי / ואחור נזרתי / קראוני ולא נערתי / ולא חרדתי מחרון אף אדני ולא יגרתי / מיום היותי על ברכי הזמן ישנתי / ובשמנו ראשי דשנתי / היה בי שפל ולא הבחנתי / ותורת אמת היתה בפי ובלבבי ולא האמנתי / קראוני ולא האזנתי / ולכן בדומיה בדד ישבתי / כי אני זקנתי ושבתי / וימי כלו בעשן / אירא מחרון אף אדני פן יעשן / ולו האזנתי דברי חכמים / חבלים נפלו לי בנעימים / ולכן מלחמה תהיה ביני ובין נפשי כל הימים / ועתה השר חבר-לי שיר וכוננהו / ומעשה ידי כוננה עליו ומעשה ידי כוננהו /



ואקום ואחבר זה השיר ואמר אליו קחהו / אשריך אם לא תשכחהו /

  

ואשא משלי ואמר:

  

צר לי בישעי מעלת הנפש



על מעלת הגוף ולא יגרתי



כי מנעורי כרשעים זרתי



ואטבעה כיום בטיט ורפש

  

איך אעדה רמה ובגדי חפש



כמה קראוני ולא נערתי



אז עם אלהים באמת ישרתי



ואהיה קונה לנפשי נפש

  

מה אמרה עוד ואני זקנתי



כלו ימי חיי בעשן כלו



על כן לבושי שק ומעטה אבל

  

רבת קראוני ולא האזנתי



לו חכמו רעי אשר סכלו



אז בנעימים יהיה לי חבל .

  

ויגש השלשה עשר ויאמר אדני יש לי אחות יפה ושמה תמר / ועיניה מבכי חמרמר / ובשרה סמר / לא עמד טעמה בה וריחה נמר / כי שדי לה המר / בסבת החשקים אשר בה ילחמו / אכזרים המה ולא ירחמו / לא יתנו שנה לעיניה ולעפעפיה תנומה / היא לשלום והמה למלחמה / זה כמה הודיעה אותם / כי קצה באהבתם / ולו היו על יד ימינה חותם / משם תנתקם / וכטיט חוצות תדקם / ולא תמצא עמהם מרגע / ולו יתמו כל החשקים לגוע / לא יוכלו עלות בהר היעלה ובקצהו לנגע / ולכן השר חבר-לה שיר חדש ובו תחרות / כי היא פסילי החשק תגדע ואשריו תכרת / והיא תשלחנו אל החשקים / אשר דלתיה דופקים / אולי מתרמית לבם יחדלו / ולא ילכו עוד אחרי ההבל ויהבלו /



ואקום ואחבר לו זה השיר הערב / וייטב בעיניו כאלו נתן בחורב /

  

ואשא משלי ואמר:

  

החשקים מה תיגעו יגע



נפשי בחשק מאנה לנגע



איך תרדפו יעלה אשר לא מצאה



נפשה בחברת חשקים מרגע



איך אזרה חיל בעז התאכזרה



לכרת אשריהם ובם לפגע



לא שוקטה אף שופטה העגבים



עד יתמו החשקים לגוע .

  

ויגש הארבעה עשר ויאמר אדני יש בעירי אשת חן חשקתיה / ומן היום אשר ראיתיה / אויתיה חמדתיה / ואהבת עולם אהבתיה / ועל לוח לבי חקקתיה / ולספר הודה נלאה כל חשק ולאיתי / כי היא פרח כל הצביות אשר בימי ראיתי / חליתי פני בוראי לחסר ממספר שנותי ולהוסיף על שנותיה / כי היא מלאה הדורים / על כל גדותיה / עד כי כוכבי השמים וכסיליהם / פשטו את מעיליהם / ואדר היקר אשר עליהם / ונתנו בעזבוניה / וצדק ונגה ירדו מכסאותם והיו שתי עיניה / והרקיע אשר על ראשי החיות היה ברד שניה / והאש תזרח ותערב בפניה / והמלאך הדובר היה לשונה / והיא היושבת במלכות הדעת ראשונה / ויד כל חכמי הזמן באחרונה / והכרובים היו שני ידיה / והרי בשמים היו שני שדיה / אם אשכחם כל ימי היותי תשכח ימיני / ואם לא אעלם על ראש שמחתי תדבק לחכי לשוני / וזה כמה תמהתי ואתמה / איך בשרי לא תאכלנה רמה / כי היא תמחץ בחץ עיניה / ולולא תרפא אותי באור פניה / ובטוב ריח שמניה / לא אוכל לקום מפניה / אמנם היא תמחץ בחץ עיניה / ותרפא בחשמלי זהרה / ועל כן ילכו החשקים לאורה / ולו תרצח היפה את כל היקום / לא יאמרו נתננו אדני בידי לא אוכל קום / ועתה השר אשביעך באילות ובצבאות / לחבר שיר כאשר יאות / ואשלחנו אל הצביה חשוקתי / כי היא לבדה בזמן תשוקתי /



ואמר אליו הנה השיר אשר לפי הענין יאות / ויקחהו ויברכני בשם אדני צבאות /

  

ואשא משלי ואמר:

  

לא יאמרו שוא עמלו בוניך



החשקים עת יחזו פניך



ברוך אשר ככה בעולמו אשר



אצר סגלותיו במכמניך



כל כוכבי שחק וחילם נתנו



החן והיפי בעזבוניך



נגה וכוכב את הדרך נתנו



שני בקשרם על ברד שניך



אשרי אנוש יביט להודך יעלה



אשרי אנוש יאחז בסנסניך



יואל אלהים חי ויוסיף מחצית



מספר שנות חיי עלי שניך



מי למדך קשת יפהפיה אשר



מחצת סגור לבי בחץ עיניך



אם תקטליני לך איחל יעלה



אצעק ולו את אטמה אזניך



אם אשכחה חשקך צבית חן ואם



אתאו ראות רזון במשמניך



או אזנחה יפעת תמונתך ואם



לא אבזרה עצמות בני חניך



תשכח ימיני לחבק עוד יעלה



או תחשביני מכלל מוניך



עפרה ואלו תרצחי כל היקום



לא יאמרו שוא עמלו בוניך .

  

ויגש החמשה עשר ויאמר אדני חנני אלהים בן והוא בשכלו אגור / ולשונו זהב סגור / הובילוהו רגליו מרחוק לגור / בשבתו תחת צלי / היה שלו בביתי ורענן בהיכלי / כאישון עיני נצרתיו / ולעשות הטוב והישר הזהרתיו / ואחרי אשר גזר האל בפרידה / ונהג בשבי נפשי השדודה / חשבתי לשלח אליו שירה / תהיה לנגד עיניו לאזכרה / והוא להזכירו שיגלה למקום תורה / ויעשה מן המדות הטובות לראשו עטרה / ואחרי אשר נתן האל מים עמקים עצה בלבו / ידלנו בעודנו באבו / ורוח אדני נוססה בו / וישים כארץ גוו וכחוץ לעוברים / ויקח מוסר השכל צדק ומשפט ומישרים / ואת אשר לרעהו לא יחמד / ובמקום הגדולים לא יעמד / וחרב שכלו על הזמן ישלף / ויפתח פיו אל דין כל בני חלף / ויזכר כי סוף כל אדם לרדת אל אבני בור / והשאול אותנו חמרים חמרים יצבר / אל ינוס הקל ואל ימלט הגבור / ועתה אדני נר המערבי / עליך השלכתי את יהבי / לחבר אליו שיר יגזר הלב לגזרים / כי שמעתי כי אחז דרך המשוררים /



ואען ואמר חייך אם הוא משורר / עלינו בשיריו לא ישתרר / כי בראותו שירי אם הוא נרדם יתעורר / וטרם ירקח כמוהו שבע פעמים יזורר /

  

ואשא משלי ואמר:

  

צבי מאס עון ואון



ואל מקום אמון גלה



ראה בלבך באר



ולך דלי וקום דלה



שפת אמת ודין בני



חלוף אהב ושוא אלה



זכר זמן יצו לך



רדה בבור בלי בלה



אמר לנפשך שחי



וטוב אמר-לך עלה.

  

ויגש הששה עשר ויאמר אדני יש בשכונתי אדם רשע / אזניו הכבד ועיניו השע / ועל פת לחם יפשע / ואם יראה עם באדני נושע / פניהם יטחן / ויאמר היוכל אל לערך שלחן / ולא ידע כי אדני צדיק יבחן / ורשע ואהב חמס שנאה נפשו / וישפך עליהם חמת אפו ואשו / ותמהתי איך יוחן רשע בל למד צדק / וכל דרכיו כמשכת חדק / ועתה אדני חבר-לי שיר על הענין הזה / ואשלחנו אל האיש אשר דרכיו בוזה /



ואען ואמר הנה שיר על זה הענין ידבר / ודלתות נחושה ישבר /

  

אלי אלי איכה יוחן



רשע ופני אביון יטחן



אף יתמרמר זד ויאמר



אם יוכל אל לערך שלחן



שוב אל עליון ובעיר ציון



תיסוד אבן אבן בחן



ידע רשע עיניו השע



כי אתה אל צדיק תבחן.

  

ויגש השבעה עשר ויאמר אדני יש בעירי שתי אחיות יפהפיות / תנהגנה לבות רואיהם כשבויות / היו כימה וכסיל לפניהן עטיות / ובנות עיש כדיות או עזניות / ושבעה כוכבי לכת למיניהם / שבע הנערות הראויות לתת להם / ובנות החן בערכם כושיות / צוארן כמגדל דוד בנוי לתלפיות / ולשונן חרב פיפיות / עברו משכיות / ובשפת שני וברד השנים / קורעות הלבבות לשנים / הולכות נטויות גרון ומשקרות עינים / וכל איש אשר הודם יחזה / יאמר מחנה אלהים זה / ויקרא שם המקום ההוא מחנים / ואחר אשר נפשנו צרה כמבכירה על האבנים / אין לתמה אם לבעליהן עשתרות קרנים / עתה אורין בר אורין / עדה גאון וגבה ולבש צדקה כשרין / וחבר עליהן שיר כפי יפין /



ואען ואמר חייך השיר יהיה כליל בהדרו / וילכו החשקים לאורו /

  

ואשא משלי ואמר:



ההולכות ומשקרות עינים



הקורעות לב חשקים לשנים



הנותנות נפש צבאים להיות



צרה כמבכירה עלי אבנים



לא אדעה במה תצודון את לבב



חשק האם הוא על ברד שנים



או על שדי חמד אשר מור נוטפות



כיום ואם הוא בעגיל אזנים



מי זה אשר יתמה בעת זאת נחזה



אם לרפאים עשתרות קרנים



תהיו לרצון לי וכל מקום אשר



אראה פניכם אקראה מחנים .

  

ויגש השמנה עשר ויאמר אדני ראיתי מתי רשעה / שכורים ביין תועה / ולצים חמדו לצון / ותחת היותם לפני אלהים לרצון / והנה ששון ושמחה הרג בקר ושחט צאן / ותמהתי ואתמה איך ימירו טובה נצחית ועומדת / בטובה כלה ואובדת / ומשוש עולם באש יוקדת / ומה טוב לאיש להיות צדקתו עומדת לעד / לערב יחלק שלל ולבקר יאכל עד / מאשר רגלו תמוט ותמעד / ובזעם אל ירכתי בור יצעד / ועתה מה בין היות איש בגלגל ערבות/ או היותו מעתד בפי החרבות / אחלי אדני לפניך / לחבר על זה השיר כפי ראות פניך /

  

ואשא משלי ואמר:

  

עד אן מתי רשעה ביין תועה



שכרו ולצים חמדו לצון



שכחו משוש עולם וגיל נצחי



לאמר הרג בקר ושחט צאן



לו חכמו אז טוב ויפיקו



מאת אלהים גואלם רצון



מה טוב ונעים לחסות בכנף



רוכב ערבות מהיות חיצון.

  

ויש התשעה עשר ויאמר אדני מצאתי כתוב לאחד מן החכמים / בתוך מוסריו הנעימים / כי מי שיתן מהונו לאחר יהיה אדניו / ואם יצטרך לו יקל בעיניו / ואם ייאש לבו ממנו / כמוהו במדרגתו יחשבנו / ואם יכבדנו יכבד ממנו / וחפצתי שהדברים האלה בשיר תחברם / וכבור והדר תעטרם /



ואען ואמר הנה השיר קחנו / ולפי הענין אין למעלה ממנו /

  

ואשא משלי ואמר:

  

תנה מהונך אל איש ותהיה



בעיניהו נגידו או אדניו



ויאש לבך מאיש זמנך



וכמוהו תהי נחשב בפניו



וכבד איש וממנו תכבד



והצטרך לאיש תקל בעיניו .

  

ויגש העשרים ויאמר אדני יש-לי רע גאה ורם / וחטאתיו לא יוכל איש לספרם / ואני שנאתיו בעבורם / לא יבוש הסכל ולא יכלם / ולא יעלה על לבבו כי לא יוכל להתעלם / ואל גמולות אדני כעל גמלות כעל ישלם / ואיך רסן תאותו לא יהיה כובש / ולא יזכר הבל נדף וקש יבש / כי אדני מלך גאות לבש / חפצתי בשיר נמרץ להוכיחו / אולי אשיב את נדחו / ואליך השר ידי פרושות / קום כי עליך הדבר לעשות /



ואקום ואחבר שיר כפי הענין אשר יבחרנו / ואמר הנה השיר אשר יכלימנו / ועל טעותו יעמידנו / לא יערכנו כספו וזהבו / שלח ידך ואחז בזנבו /

  

ואשא משלי ואמר:

  

היגאה וישלם אנוש דך ונכלם



ועל טוב ועל רע גמול לו ישלם



וירבה אשמיו וחטאת עלומיו



כחול ים וספר אמת לא יכילם



וישכח מעודד ענוים ודרך



רשעים יעות וימלך לעולם .

  

וישמח האיש ויעמד מזעפו / וישלח ידו ויחזק בו ויהי למטה בכפו /



ויהי כאשר תמו כל האנשים לשאל איש איש את שאלתו / ומצא כל אחד מהם צרי אל מחלתו / הודו וברכו לאל אחד ואין שני / ויאמרו אלי החייתנו נמצא חן בעיני אדני / והחסד אשר עשית עמנו איש לא שער ולא ידעו / ונהיה אנחנו ואדמותינו עבדים לפרעה / אשר השר שבחרך לנחלתו וחבלו / ואשרי העם שככה לו /



וילכו שמחים וטובי לב איש לאהלו / ויאמר השר באמת השבעת נפש האנשים בצחצחות / כאשר בענין השירים דברת צחות / ואותך לארך ימים ושבע שמחות / יפקד אדני אלהי הרוחות.
