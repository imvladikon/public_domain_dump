


	
	מחברות עמנואל
	



מחברות עמנואל – מחברת שבע-עשרה / עמנואל הרומי


אמר המחבר: כאשר שבתי עם השר מבית הגבירה ההדורה / אשר לא יתן סגור תחתיה ולא ישקל כסף מחירה / היה לבב השר כמו תנור בוערה / נלהב מחשקה ובוער / זעקת שבר יעוער / וכמוץ מגרן יסוער / ובכל מחמדי הזמן גוער / ויהי היום ויאמר אלי השר אשביעך בברית האהבה אשר בינינו / ללכת אל בית הגבירה שנינו / כי מחשקה נלהבתי / ואהבה עזה אותה אהבתי / ושיריה מצאו חן בעיני / ושמתי בלבי לשנס מתני / ולעורר רעיוני / לרגל המלאכה אשר לפני / ולחבר שירים על אהבתה אולי תשא פני / וגם אתה הכון והכן לך שירים צרופים בעליל היפי / ומזקקים שבעתים מאין דפי / יהיו שקולים במאזנים / נחמדים להשכיל ותאוה הם לעינים / יבקעו לב-הגבירה לשנים /ונגלה אליה מצפון לבנו / ואש החשק הטמון בחבנו / אולי תקל מעצבנו /



אז אמרתי אל השר שים לדרך פעמיך / והט שמיך / והוצא פניניך ולשמיך / וגנזי ברומיך / מבין אסמיך / ואחריך לבבי בעממיך / ועל הרי הצביה תרים נסיך /כעמי כעמך כסוסי כסוסיך /



אז קמנו והלכנו אל בית הצביה היפה / פוריה וענפה / לארות יערת דבשה וצופה / ומצאנוה בעד החלון נשקפה / תחפיר השמש בצאתו / ומאן השמש להראות בפניה יפעתו / ובעלה חרפת אדם ובזוי עם / היה נצב על אם הדרך כאיש נפעם / ועמה דברי ריבות מדבר / אבל וחפוי ראש כאיש אביו מקבר/



ואמר אל השר ארור האיש אשר זה החבור חבר / זה הוא בעל היפה פתע ישבר / ועתה חילים נגבר / ודלתות נחושה נשבר / וערם נערים / ונעשה עצמנו כמתנכרים / וכאלו לא נכירהו נהיה / מאדות הגבירה בפניו מדברים / ונחשבהו לחשק / ולאיש אשת אחרים גוזל ועושק / ונאשרהו אשר יחשק גבירה כזאת/ ונדבר לפניו דברי חשק בחרוזות / ונעשה אל הגבירה רמיזות / לראות אם הוא כוסה קלון או יענה עזות /



ויאמר השר עת לנסות / טוב הדבר אשר דברת לעשות /



ויהי כאשר קרבנו אליהם ויאמר השר שלום לשר החשקים / ושלום לשושנת העמקים /



ותשחק הגבירה ותאמר / ועליכם שלום החשקים החשוקים / אשר בלב היעלות חקוקים /



ויען בעל האשה הנרצחה / אחרי אשר התאנח מלבו אנחה / ויאמר ראיתי היום כי חשבתוני לחשק / ולאשת אחרים גוזל ועושק / אינני כאשר חשבתוני / וזה לי האות כי לא הכרתוני/



ויאמר השר אל תחרד הדוד ואל תשם / ידענוך לחשק מתמול שלשם / ואין לתמה אם חשקת במלכת השמים / כי גם אנחנו חשקנוה לא יום אחד ולא יומים / ולולא התמהמהנו כי אתה שבנו זה פעמים / ועתה הנה אנחנו פה שלשה חשקים / בכנף היעלה מחזיקים / בזאת נבחן / אם בעיני הצביה נמצא חן / יחבר כל אחד ממנו על חשק הגבירה שיר נחמד / אשר כל הבא אלי ועמד / והיה האיש אשר יהיה שירו כליל ביפיו ובהדרו / נניח לו הגבירה / כי לו משפט הבכורה / ואני אשים פני בראשונה להלל יפעתה / אולי יישר בעיניה ואשכון על הריה ועל גבעתה /

וישא משלו ויאמר

לא האמנתי אומרים לי אמש



יש יעלה תלעג עלי ירח



עד אחזה האומרה לשמש



עת יעלה מזרח עלה קרח.

ויהי בהשלימו השירה / ויאמר הנה הללתי הגבירה ההדורה / ועתה יאמר כל אחד מכם שירו / ונראה למי תפארת השיר והדרו / ואען ואמר חי אני אם בשיר נרצה הגבירה קרוב צדקי / אענה אף אני חלקי / כי גם לי שירים כברקים ירוצצו / וכעין נחשת קלל ינוצצו / ולבות האבן בנקלה יפוצצו /

ואשא משלי ואמר:

עפרה תשקיף מן האשנב



ובעיניה יונה או תור



לבי תציב כמטרה



תרעיש ספיו הך הכפתור



כעת חצה תורה צדה



תאמר תשמע החלום לפתור.

ויהי בהשלימי השירה / ואמר אל בעל הגבירה / אמר נא החשק גם אתה שירך / השמר והשקט אל תירא ולבבך אל ירך / כי אם תגבר ידך עלינו בשירה / נניח לך חשק הגבירה/



ויאמר האיש חייכם החשקים הנעימים / בוקר אנכי ובולס שקמים / אינני חשק ולא בן חשק / ולא אשת אחרים גוזל ועושק / אמנם דבר סתר היה לי אל הגבירה / וחלילה לי מדבר חשק ודבור סרה / כי החשק גנוהו החכמים ואמרו בעדו שהוא מפתח כל האשמים / ורסן מתעה על לחיי עמים / ולבעליו לשון סתר ופנים נזעמים / והדורכים דרכו ילכו קדורנית /יגלו ערות אביהם ופניהם אחורנית / והיא הגדולה שבמחלות ונקראה מחלה רוחנית / ושרשה במח נטעת / והיא מכוס התרעלה קבעת / אין טוב בדעת לחשוק אכן טוב לחשוק בדעת /

ואמר המשורר:

החשק מחלה רוחנית



שרשה במח נטעת



אין טוב לך בדעת לחשוק



אכן טוב לחשוק בדעת.

ויאמר השר ולמה תעלים / חשקך זרזיר מתנים / ואתה חשק ממנו פי שנים / ומי לא יחשוק הבת ירושלים / אשר גזלה נוגה וצדק מבין כוכבי השמים / והיו בה לעינים / וזה כמה עמדתי משתאה / ממראה עיני אשר בה אראה / כי אביט השמים והנה נוגה וצדק אינם / לא ידעתי אם גלו בעונם / ושאלתי הוברי שמים אם ידעו מאדותם / ויאמרו כי בפני הגבירה ראו אותם / מפתחות פתוחי חותם / ומשבצות זהב במלאותם /

וישא משלו ויאמר:

אפקד צבא מרום ואראה כי



נפקד מקום צדק ואור נגה



ואשאלה כל הוברי שחק



מי זה ואיזה הוא אשר שגה



לגנב כהיום ככבי מרום



ולהעמיד ארץ עלי תוגה



ויאמרו לי דוד שאלתך



למאד חקרנוה להשיגה



לא נעדרו צדק ונגה כי



על חוג אדמה זהרם יגה



אך ירדו המה ושכנו ראש



עפרה בשושן הוד ונוי סוגה



שור עין ימינה תחזה צדק



שור עין שמאלה תחזה נגה.

והיה כאשר השלים השר שיריו / באתי אחריו ומלאתי את דבריו / ואמרה חייכם החשקים גם אני ככם אדבר / ודבר חשק על אהבת הגבירה אחבר / כי הגבירה ידידות נפשי / ואשים חשק אהבתה /
עטרת לראשי / ולו תהיה שמאלי תחת לראשה / ובימיני אחבקנה ותהיה נפשי קשורה בנפשה / תהיה לי כל הנעימות למורשה / אינק ואשבע משד תנחומיה / ואלחם במנעמיה / ואארה מורה ובשמיה / ואם גנבים יבואוני ושודדי לילה לריק לא יגעו / יגנבו דים וישבעו / ולא אמר סורו סורו אל תגעו /



ואשא משלי ואמר:

לו יד שמאלי תהיה תחת



ראשך צבית חן ופי אל דד



אינק ואם שודד יבואני



לא אמרה כי אהלי שדד.

ויהי כהשלימי השירה / ויחר אף בעל הגבירה / ויאמר אוי לכם שעירי עזים / אנשים ריקים ופוחזים / ושוכבים וחוזים / אשר ארחותיהם עקשים ובמעגלותיהם נלוזים / רואה אני את פניכם לחשוק נחפזים / ודרכי הנדיבות בוזים / כאלו לא עמדו אבותיכם על הר גריזים / הלא ידעתם כי לגבירה גואלים ואחים / והיא כשושנה בין החוחים / ואיך תחזו לה משאות שוא ומדוחים / ולה בעל חותם תכנית / ואיש יגע בהם ימצא ברזל ועץ חנית / יחרשו על גב החשקים ויאריכו מענית / וישיבו צל המעלות אחורנית / ומה לכם לרחם רחמתים / והעזות על פניכם כאמתים / שובו לאהליכם ותשכבון בין שפתים / אמרתי אך דבר שפתים / ומדוע נגשתם אל החומה/ ועתה עצה וגבורה למלחמה / ומן התמה עליכם בני עוננה / כי יחשבו אתכם הרואים לאנשי שכל ותבונה / ואיך תתפתו אחרי אשה תראה יפה כלבנה / ועל דרך האמת תשכן עליה עננה / הלא הגד מראש לכם כי בלבנת היעלה אין לבטח / כי כמה שחורות שבו לבנות אחרי הטח / והם למצודים וחרמים משטוח /

ואמר המשורר:

דודים בלבנת יעלה אם לא



ידעתוה תמול אל תבטחו בטח



כי ראתה עיני שחורה מ-



שחור שבה לבנה אחרי הטח.



אשר עוד ראיתי במחשבי / ונתון אל לבי / כי כמה מן הכעורות היו לאדמת דמן / ייפה אותם השמן / ותחת היותם מרוח קדים שדופות / ושפחות חרופות / תחשבנה בעת ישמינו מן היפות / ותהיין כאריות טורפות / ועפר ארץ בראש החשקים שואפות / ואמר אחד מאנשי המועצה / בדרך משל ומליצה / כי בלכת אנה ואנה אריסטו מצא השמן מתהלך



לאטו / וישאלהו לדעת משפטו / ויען כי כה מבטו/ליפות הכעורות / ולהלבין השחורות / וכאשר יחשבו החשקים כי חשקו צורות הדורות / אז הם רוכבי אתונות צחורות /

ואמר בשירו:

תמול שלשום שמנים וחלבים



פגשתימו בנחל הבשורות



שאלתימו ידידים אל פשטתם



השיבוני לקשט הכעורות.

ויש לכל חכם לב להנצל מערמתן / ולהושיב קשתו באיתן / פן חמס הנשים יכסם ושד בהמות יחיתן / כי חמתן מרבה מצלתן / וכבר אמרו החכמים / וגלוי לכל העמים / כי האשה מצודים וחרמים / והיא סבת כל מרי נפש ואיבה ומריבה / ולמות ולכל תלאה עלה וסבה / טוב לפני האלהים ימלט ממנה וחוטא ילכד בה / ואמר החכם האשה סבת כל רע למגדול ועד קטן / והיא מצודה פרש אותה השטן /

ואמר המשורר:

מוצא אני מר ממות



הנשים הוביש מבטן



נחנו צידה הם כמצודה



פרש אותה יד השטן.

ולכן הירא את דבר אדני מעבדיו / ויפחד מאשו ולפידיו / ישמר מחצי עיניה / ויעצם עיניו מראות פניה / ואם יבהילוהו אוריה ותמיה / יהפך אותה ויתגלו אליו מומיה /

ואמר המשורר:

חמה וקצף אף צבי החן שפך



על יעלה וגער במנעמיה



אם לבך יטה עלי חנה הפך



אותה ויתגלו לך מומיה .

וכמה מן החכמים המחכמים / ומזרע המלוכה ומן הפרתמים / אשר היו על לוח לב הזמן חותם / והחשק עצם ראותם / עד טח מראות עיניהם מהשכיל לבותם / ולא הבינו לאחריתם / וימירו בקלון את כבודם / ומשנה קצף לקחו בידם / והניחו על קרן הצבי צבי אשר מאין ליש העלם / ובאהבתו ובחמלתו הוא גאלם / וינטלם וינשאם כל ימי עולם / רוכבי אתונות צחורות היו למו לברות / ונסעו בסבתם מהר שפר ומחצרות / ויחנו בקברות התאוה ובמוסרות / ואמר אחד בשירו / בראותו כי החשק במאסרו אסרו /



אני בן חכמים ומבין עלומים



ובן גור אריות ואמי לביה



והשע ועצם מאורי לבבי



לבלתי ראות הוד צבי יה צביה.

ואמר חכם אחד שאלוהו אנשי עדתו / על אדות חכמתו / ומעשה תקפו וגבורתו / דעו אפוא אחי / כי סבת חכמתי ועלת שבחי / ואשר החכמים ישקדו על דלתותי / לשמור מזוזות פתחי / ותהי להם לאזכרה ריח ניחוחי / והיה כל הנשוך וראה אותי וחי / הוא תחת היות איש חשק לנגדו הצביה מעמיד / שויתי אדני לנגדי תמיד /

וישא משלו ויאמר:

לשאל במה נעלית עדי כל



נשוך הזמן יחזך והיה



עניתיו תמורת להציב לנגדי



ראות מן צביה הציבי צבי יה.

ואין לתמה אם האיש יחסר הודו ויפעתו / ויהיה רוח אלהים רעה מבעיתו / בהתחברו עם האשה לרעתו / ושחין אבעבעות עליו פורח / ויהי מכאב לבו מר צורח / כי הנה השמש הזורח/ הגדול על כל צבא השמים בהכרח / לא ילקה רק בהתחברו עם הירח /

ואמר המשורר:

אל תתמה על איש דודאי



חשקו כיום נתנו ריח



אם עת יתחבר עם עפרה



זיו הודו מנו בורח



לא ילקה השמש רק עת



יתחבר עם הירח.

ולכן חדלו לכם מן החשק אשר מפיכם יוצא ושב / כי במה הוא נחשב / ותנו לכם מרגוע ונפש / ולבשו לכם מן השכל בגדי חפש / פן יפגעו בכם אנשים מרי נפש / ואספתם את נפשכם / ונפש בית אביכם /



ויאמר השר הזאת חשבת למשפט עלינו / ליראנו ולהבהילנו / באמרך כי יש לגבירה בעל חותם תכנית / חי אני הנני מתנדב לגזל מידו החנית / וילך מפני קדורנית / וגם לגבירה אחים שבעה / ארדף אותם מהר אל הר ומגבעה אל גבעה / ואקח לי הגבירה על אפם ועל חמתם / ואשימנה על לבי חותם / ובחפץ גדול הייתי בעלה מכיר / כי הייתי מכה אותו בחנית ובקיר / או חצי מדמו אשכיר /



ויפן השר אל הגבירה / ויאמר הנני נשבע באהבתך היקרה / וביפעתך הטהורה / ובנגה זרחך הנטוי על ראשי החשקים כעין הקרח הנורא / ובעיניך אשר כברקים יהלוכו / ובגבותם אשר קשתותם ידרוכו / ולבות החשקים ימשוכו /כי לא אפרד ממך עד אשר אל חדר הורתך תביאיני / או בעלך חותם תכנית תראיני /



ותצחק הגבירה ותאמר חייך השר אם תראנו תתחרט / כי הוא איש צרוע וראשו ימרט / וכל עומסיו שרוט ישרט /והוא נתעב ונאלח / כעוף נודד קן משלח / חץ כבדו יפלח / ואם תחפץ לדעת איך נסעתי מיטבתה אל אדרעי / הוא האיש המדבר עמכם זה דודי וזה רעי /



ויקצף האיש על הגבירה / וחמתו בו בערה / ויאמר אי פרה סוררה / אשת כסילות ואשת מרי / אשר תלכי עמי קרי / כדבר אחת הנבלות תדברי / יכך אדני בשחין רע ולא ימיש מעשות פרי / אמרי נא עוננה מה חזית / כאשר קדשי בזית / ואהבת משכבם יד חזית / חי אני כי לולא אשר עלה על פרי הדרך חזזית / ואחרי מראה עיניך התפתית / אביך אמורי ואמך חתית / אז ראית צורתי האמיתית /

וישא משלו ויאמר:

בזית צורתי שית זונה



מראה עיניך התפתית



לולי כעור צורתך אז



ראית צורתי האמתית.

ויאמר השר והוא מביט אל הגבירה /אשר נפשו בנפשה קשורה/ אי צורה נעימה והדורה / זה הוא אשר קורא עצמו חותם תכנית / אשר פניו כפני התענית / ועל גבך יחרש ויאריך מענית / אי הילל בן שחר איך נפלת משמים/ ותהיי לאיש כמר מדלי וכשחק מאזנים / חי אני לו היית דלתי דופקת / ועל דדי מתרפקת / אז נודע לך כי בהמה על צורת איש את חובקת /

וישא משלו ויאמר:

את מי תחבקי עפרת חן



על מי זה את מתרפקת



לו היית לי נודע לך כי



כעת “חיה” את חובקת.

ויהי כהשלימו השירה / חמתו בו בערה / ויצעק צעקה גדולה ומרה /ונהה נהי ונהיה / אוי ואבוי ואויה /אמר שדוד נשדנו / חבלי שאול סבונו / ועוד ידם נטויה / בבית ישראל ראיתי שערוריה / כי פרה אדמה תמימה בשכלה / אשר על עליה לא עלה / ואין על עפר משלה / נפלה כנפל לפני בני עולה / תסלד בחילה / ונפשה מרה לה / ולא חס עין אנוש עליה לחמלה / ובעלה ארי במסתרים ודב אורב הוא לה /

וישא משלו ויאמר:

איך טחו מהשכיל לבות



איך עיני רואים תשעינה



כי ראיתי שערוריה



כי פרה ודב תרעינה.

ויהי טרם כלה השר שירו / ואני באתי בדברו / ואמר אחלי לפניך השר אל תמנע ממני לחם חקי / ואענה אף אני חלקי /

ואשא משלי ואמר:

אתמה על כל חשקי חשק



איך בחיים לא יקוצו



כי ראיתי שערוריה



דבר עם הפכו נקבצו



כי ראיתי פרה ודב



תרעינה יחדו ירבצו.

ויהי כראות האיש כי הרחבנו עליו פה / ולמחלתו אין ארכה ומרפא / היה כאיש נדהם ועלפה / כאלו נדקר בחרב ונספה / לא יפתח פיו כאלם / ומפני קצפי וחרון אפי מתעלם / והגבירה בצרת בעלה שבעה / ואז ימלא שחוק פיה ושפתיה תרועה / ובעלה מצרתו צועק חמס / רשע יראה וכעס שניו יחרק ונמס /



ויהי אחרי כן וישא האיש את עיניו וישם אל השר פניו / ויאמר נפלאתי עליך השר /בעל חכמה ומוסר /בידעי מעלתך / ואת יקר תפארת גדולתך / וידעתי כי את אדני אלהיך תירא ואותו תעבד / ולך אשת חן תתמך כבוד / ואיך באשת אחרים תחשק /הטוב לך כי תעשק /והיה לך לחמל על יחס אבותיך/ ולגעור בחשק אשר לא מבריתך / למה תמות בלא עתך / שובה השר מזדון לבך ועורה משנתך / ופקד בצדק הון אוצרותיך / והיה לך למשיב נפש ולכלכל את שיבתך / פן מה תעשה באחריתך /ואמר החכם לבנו בני לך יחס מכבד / ואם תשמרנו בטוב המעשים לא יאבד / ולך הון עתק בצדק תפקדנו / ודע אם לא תפקדנו תפקדנו / 1



וישא משלו ויאמר:

בני יחסך מכבד תשמרנו



בטוב המעשים פן תאבדנו



והונך רב בצדק תפקדנו



ואם לא תפקדנו תפקדנו.

ויש לך השר להבין לאחריתך / ולהוסר ברעת זולתך / ולא יוסרו אחרים ברעתך / ואמר החכם בספר המוסרים / לץ תכה ופתי יערים / ודע כי האלהים יתעלה גזרה חכמתו / אשר איש שלא יוסר ברעת זולתו / יוסרו אחרים ברעתו/

ואמר המשורר:

קחה מוסר והוסר בזרים



ואל יוסרו בך הבערים



ודע כי איש באחר לא ייסר



ייסר בו אלהים חי אחרים.

ועתה לך איעצך השר / העבר חשק מקרבך ועדה עדי חכמה ומוסר / ולבש תמיך ואוריך / והעבר רעה מבשרך / והלכו גויים לאורך / ואם את הדבר הזה תעשה וחקותי תלמד / ואת אשת עמיתך לא תחמד / וצוך אלהים ויכלת עמד / והימים הראשונים יפלו כי מה שעבר חלום / וגם כל העם הזה על מקומו יבא בשלום /



ויען השר ויאמר ידעתי כי לי אשת חן וכבוד תתמך / אמנם בכל יכלתי על אשת אחרים אסמך / ולא אחוש בהביטי אל מקום נמך / וזאת הגבירה מיום ראיתיה אויתיה חמדתיה / ואהבת עולם אהבתיה / ועל כן חסד משכתיה / ועל גרגרותי ענק ענדתיה / זאת מנוחתי עדי עד פה אשב כי אויתיה / ואשר תאמר איך אחשק באשת אחר / דע כי אני מבקש רצון וטוב שוחר / אמנם אשת בן מחפיר ומבאיש / לא אחשב לאשת איש / ואתה אחרי אשר עונתה תגרע / והיא לא תצמיח ולא תזרע / ראוי שיגע בך רע / ואשר יעצתני בן מנבל / לחלל ברית החשק ולחבל / בזה עצתך לא אקבל / וידעתי כי לי יחס נכבד / ולא אירא כי בחשק הגבירה יאבד/ ואם תשאלני עליו אני מגזע ישי / ואם עצתך עצת אחיתופל אתי עצת חושי /

וישא משלו ויאמר:

דודי שאלני עלי יחשי



ואמרה בן עבדך ישי



אמר צבי קח לך עצת לבי



ואענה טובה עצת חושי.

ואיך לא תהיינה ידי אמונה / לחשק ביפה כלבנה / ומקטרת מור ולבונה / חי אני כי מיום הכרתי העדינה / וראיתי דודאי חשקה נתנו ריח / ברכתי על אורה הזורח / ולא על חדוש הירח /

וישא משלו ויאמר:

בידעי כי הלבנה שם משתף



לירח ואל עפרה לבנה



אברך על צבית חן בחדוש



מאורה ולא בחדוש הלבנה.

ומן הדין היה לך להחזיק אלי טובה נאמנה / כאשר הייתי עם הגבירה באמנה / כי אאהב את אשר תאהב ומשנאיך אשנא / והיה לך לשנא אותי אם אשנא אוהביך / ואגעל בקרוביך / והנה רצונך עם רצוני נתיחד / עולות בקנה אחד /



ויאמר האיש אל תחשב השר כי הגבירה אהבתיה / כי תכלית שנאה שנאתיה/ ולולי חמלתי על כבודי פן יחשב אלי לפחיתות / כתבתי לה ספר כריתות / וחי אשר קנה לבבי וקם לו לבד לצמיתות / בשנאתי אותה אתאו להיות מעוך וכתות /

וישא משלו ויאמר:

בחי האל אשר קנה לבבי



ולבי קם לבד לו לצמיתות



בשנאתי לשוכבת בחיקי



אני אתאו היות מעוך וכתות.

ויאמר השר ואם יגזר המזל ותמות אשתך / התלכד צביה אחרת ברשתך /



ויאמר האיש חייך השר אם יאמרו לי בני עמנו / נשא אשה ומלוך תמלך עלינו / לא חפצתי לקחת מלכה או נסיכה / וקדם חפצתי להיות כרות שפכה /

וישא משלו ויאמר:

לו יאמרו עמי נשא אשה



כיום ועלינו לכה מלכה



אתאו היותי נעבד בלתה



או להיות עמה כרות שפכה.

ויאמר השר הלא ידעת כי לא טוב היות האדם לבדו / ולכן נבראת האשה לצדו / להיות לו עזר כנגדו / וכמה מן הנשים/ יעלימו מומי האנשים / וישביעו בצחצחות הנפשים / ותהיין כאניות סוחר וישמרו הרכושים / ואמר החכם כאשר שאלוהו / איזה טוב לבני האדם אשר יעשוהו / ויאמר כי האדם יחיל טובו / בשכל יהיה בו / ואם אין זה יתהדר במסכת זהבו / ואם אין זה יקח אשה כשרה תעלים נגע לבבו / ואם אין זה ישתק ויכסה זדון לבו בקרבו / ואם אין זה הקבורה תהיה סכה לראשו ורבו /

ואמר בשירו:

שאלוני ומה זה טוב באדם



השיבותי בשכל יהיה בו



ואם אין זה עניתים העשירות



להתהדר במסכת זהבו



ואם אין זה עניתים החברה



כשרה תעלים מומי לבבו



ואם אין זה עניתים השתיקה



לכסות את זדון לבו בקרבו



ואם אין זה עניתים הקבורה



תהי סכה עלי ראשו ורבו.

ויאמר האיש אמנם ידעתי כי הנשים לעזר נבראו / אמנם מעטות על זאת התכונה נמצאו / וכמה קשה לסבל חברת הרעות מהם / לא מהם ולא מהמונם ולא מהמהם / ואמר החכם בזאת יבחן האיש נפשו החכמה / לדעת כי הגיעה למעלה נכבדת ורמה / שיוכל לסבל חברת אשת הזמה / כי האשה הרעה / תחליף שערה ולא טבעה /

וישא משלו ויאמר:

בזה יבחין נפשו איש ש-



יסבל חברת אשה רעה



המשלוה לזאב ערב



תחליף שערה אך לא טבעה.

ויאמר השר אין לתמה אם חשקנו חשק עצום היעלה / כי שמענו אומרים כי עודנה בתולה / ורבים חשקי בתולה מחשקי בעולה / ולמעלת הבתולה נתנה האל בשמים / בין ארי ומאזנים /

ואמר המשורר:

ברח דודי ברח מעיר פרוצה



וקח נערה בתולה וחרוצה



וקח מפי לך באור נערה



ונערה מעפרה והקיצה.

ומן התמה מן הגבירה איך בחיים עודנה /כי נשאתה ותשליכנה/ וכרחוק מזרח ממערב תרחיקנה / וישועה רחקה ממנה / ולכן אשה תארש ואיש אחר ישכבנה / ולו למדת והיו תאמים מלמטה / כל אנחתה השבתה / הלא ידעת כי לא ילקה הירח או אורו לא יחצה / רק בהתרחקו מן השמש מן הקצה אל הקצה /



וישא משלו ויאמר:

עפרה עת עם עפר תשכן



תאיר תזריח ותרצה



אכן עת נוד דוד חשקה על



נודו גם לבה ימצה



תדמה לסהר לא תחשך



כלה או אורה יחצה



רק עת תרחק מן השמש



מן הקצה אל הקצה.

ויאמר האיש חלילה לך השר מחזות מהתלות / ולומר כי למעלת בתולה הושמה בשמים בין המזלות / והנה אם תאמר כן בבתולה / מה תאמר בסרטן ועקרב אשר אצל גבולה / התאמר כי בעבור מעלתם הושמו / בין ראש כוכבים כי רמו /



וישא משלו ויאמר:

בתולה יחשקו חשקים ויאמרו



למעלתה זבול היה זבולה



אשיבמו ראו סרטן ועקרב



בצדה הם למעלה לא מעלה.

ויאמר השר חי אני כי דבריך לא יועילוך / ומשליך והתליך לא יצילוך / הגבירה הזאת לא לך תהיה / ואשר יקום אלינו אם בהמה אם איש לא יחיה / מות יומת דמיו בו יהיה /



ויהי כראות הגבירה / כי בעלה ידבר עליה סרה / כאלו היתה לזנונים הרה / ותאמר בלבה אחרי אשר בי ימעל / ובחשקי יגעל / בור כרה ויחפרהו ויפל בשחת יפעל / ותשם אל לבה להגדיל עצבו / עד יהיה גונח מלבו / ותאמר אלינו שמעוני החשקים הנעימים /כי אתם נזר הימים /בעלי זקן בא בימים / בחשך בא ובחשך יתעלם / ויצוריו כצל כלם / חלפו עם אניות אבה / ולחשק לא ישמע ולא יאבה / ולי סחורות יפות ונמרצות /צבי הם לכל הארצות /גלות עליות וגלות תחתיות/ עברו משכיות / ובעלי מך מערכם / והוא בן לא חכם / ואתם שני חשקים / על דלתותי מתדפקים / אחלק גופי לכם לשני חלקים / הנה גופי אשר דמתה לתמר קומתו / והוא כחתן יוצא מחפתו / האזור יגבל אותו / קח אתה השר בראשונה / ויד רעך תהיה באחרונה /



וישא השר עיניו וירא גופה נחמד למבטיו / וכל אשר בה שוכן לשבטיו / כלליו ופרטיו / וכראות השר נטע נעמן / דלת ראשה כארגמן / ושפתיה שפת כוס פרח שושן מעשה ידי אמן / ויפי העינים / וברד השנים וכפתוריה ופרחיה / והדוריה ושבחיה / וצוארה כמגדל השן / ויאמר ארור האיש אשר עם זאת הגבירה ישן / ובראותו חזה התנופה אשר הונף ואשר הורם / ויאמר זאת הארץ אשר נשבע אדני לאברם / ויבחר לו המנה העליונה / ויאמר כי לולי היותה המנה הנכבדת והעדינה / לא היתה למשה למנה /



וישא משלו ויאמר:

מה אמרה עת אחזה ממעל



יפעת תמונתך אשר זורחת



אביט אלי ארץ ושמה תאות



נפשי כאש תבער ומתלקחת



אמנם לגדל מעלתך יעלה



תרגז יחידתי שאול מתחת.

ויהי הוא כלה לדבר וירא בעל היפה / כי הגבירה תתנדב לנו בגופה / ויאמר אבדה האמונה / ואין לה שרש וענף / כי הארץ חנף תחנף / ויקצף בלבו ויתאנף / ויחשב בלבו ערמות/ להנצל מיד מהומות / ויאמר אלינו ורעיוניו פוחדים ורוהים / שמעוני וישמע אליכם אלהים / היום הזה חשבתוני לחשק כמוכם / ויצא דבר מלכות מלפניכם / גזרתם אמר ויקם לכם / אמרתם יחבר כל אחד ממנו שירה / על אדות הגבירה/ והיה האיש אשר במלאכת השיר תגבר ידו / נניח הצביה לו לבדו / ועתה אחרי אשר המזל פה הביאני / השמני והלאני / ולראות אשר לא דמיתי לראות הראני / הנני למלאת מצותכם ארצה/ ותהיו שניכם וכל החשקים מקצה / ונחבר שירים מאדות זאת הקדשה / ואם לא אגבר עליכם בשירה חדשה / בשפה ברורה בנעימה ובקדשה /תהיה על לוח לב הזמן חרושה/ ונתתי אותה לכם למורשה / אך ידע תדעו כי אני כאדם עשוק ורצוץ / אגנה הברוש ואפאר הנעצוץ / כי חשק לא ידעתי ולא עבר מעולם בי / אך הניפותי לעולם עליו חרבי/ והכרתי אותו מקרבי / אשר לא צויתי ולא דברתי ולא עלתה על לבי / אמנם כאדם אנוס / מן הפחד את הפחת אנוס / ועתה הגישו עצמותיכם אנשי הזמה / וקומו ונקומה ונעלה עליה למלחמה / ויעבר עלי מה /



ויהי כשמע השר ניבו / עלז לבו בקרבו / בחשבו כי ישיב צור חרבו/ ויאמר זה היום עשה אדני נגילה ונשמחה בו / ויפן אל הגבירה / ויאמר רני ושמחי צורה ההדורה / כי היום הזה יום בשורה / כי ידנו במלאכת השיר תגבר / ושיריו תעו במדבר / אין חפץ בם לא לזרות ולא להבר / ועתה מה לתבן את הבר / גילי ושמחי כי נשבר שבט מכך / ותחסי תחת כרוב ממשח הסוכך / יהיה פריו מתוק לחכך / כי הנני בא ושכנתי בתוכך /



ותאמר הגבירה חייך השר אף על פי שצורתו כעורה / ופניו כשולי קדרה / ישלח אדני בו המארה / לו שפה ברורה / ושירתו חזקה ונמרצת / מפניני המאמרים מקבצת / ואם לאשמת העם יגבר עליכם בשירתו בן הזונה / תהיה ידי על התחתונה / ומה תעשה נפשי הענוגה / אם תהיה על זקן ממרה למנה /



ויאמר השר אל תראי חבצלת השרון / כי אחבר עליך שירים יהללו בכל פה וגרון / ותענדי אותם ענקים לגרגרותיך ושמתם בארון /



ותאמר הגבירה אלהים יענה את שלומכם / וירצה אדני לתתי להלך עמכם /



ויאמר האיש איך יהיה ענין שירינו בדברי יקר או בזלות/ בדברי חכמה או בדברי הוללות /



ויאמר השר נפליג לדבר בנבלות / עד אשר יחשב לנו לשפלות/ ויהיו שירינו הוללות וסכלות /



ויאמר האיש ומי יהיה לחטאת ראשונה /



ויאמר השר אני אגאל היום מבין החוחים השושנה /

וישא משלו ויאמר:

עפרה הסבי עיניך



מעיני סער לא נחם



לבי תציבי מטרה



מחציך יום יום נלחם



עלי נפשי תשתוחח היא



מתלקחת תוך אש פחם



עד אן תסתירי פניך



תורידי לארץ נצחם



אחלי עפרה הריקי לי



ברכותיך אז אתנחם



ברכות אם רובצת תחת



ברכות שדים ורחם.

ויהי ככלות השר שירו באתי אחריו / ומלאתי את דבריו /

ואשא משלי ואמר:

עפרת חן גפן פורחת



מפניך לבי נחת



הראיני נא את מראיך



כי על חשקך מראי משחת



נדמה חשקך אל תוך קרבי



ברד עם אש מתלקחת



השמעיני רעיה קולך



ופדיני מרדת שחת



הביאיני אל גן עדנך



הגמיאיני יין מרקחת



ממגד שדים מעל



ומתהום רובצת תחת.

ויהי טרם אכלה לדבר ושירי עודנו בחכי / ויאמר האיש אל תאחרו אותי ואדני הצליח דרכי /

וישא משלו ויאמר:

מה יפו דודיך עפרה



אור לחיך על כל אור מלך



נפלתי במצוד אהבתך



כנפל דגים לפני שלך



מיראת סערתך אנוס



מחמתך אל הוד צלך



ארחץ סותי עם דם דמעי



אשטחנו אל אור הלך



בחרתי הסתופף כיום



להיות עמך לשכב אצלך



רעיתי אל תעצרי לי



לרכב כי אם אמרתי לך .

ויהי כהשלימו השירה המהללה /נתן עלינו בקולו קול המלה/ ויחרד השר חרדה גדולה / ויאמר זה קול ענות גבורה / בפעם הזאת לא נצחנו הגבירה /



ואען ואמר אל השר אין לנסות עוד בשירים / כי לו פניני המאמרים / ויהלומי הדבורים / הבה נתחכמה לו בסודות / ונחוד לו חידות / כי אם גברה ידו בשיר קצר שכלו / אולי יפתה ונוכלה לו /



ויאמר השר מה מאד היטיב לדבר זה הנבל / אשר לקברות יובל / ובשרו עליו יכאב ונפשו עליו תאבל / רואה אני כי כשל כח הסבל / ועתה חודה לי חידתך ותרחיב לי בצר / כי ידי מהושיע תקצר / ומזמה ממני תבצר / ולבבי כח לא עצר/



אז לבשה אותי רוח קנאה / ואמר אל בעל הנאה / אחודה נא לך חידה / אם יכול תוכל להגידה / נניח לך הגבירה בכבודה / ולא נדבר עוד מאדותיה / ולא נשחק עוד על משבתיה /



ויאמר האיש השבעו לי כיום / ונשבע לו וישמח האיש כי רב חילו / ויאמר שמע אלהים בקולי / אך עשרתי מצאתי און לי/ הנה אשתי אשר תחמדנה / חודה חידתך ונשמענה / ואם אעמד רגע ולא אדענה / קחו לכם אשתי / אשר לקחתי מידכם בחרבי ובקשתי / והתעללו בה בפני לבשתי /

וישא משלו ויאמר:

אין לי יריבון עלגי שפה



נרפים ולצים חמדו לצון



כי נחשבו לי כל נאות השיר



לנוות כרות רועים וגדרות צאן.

ואען ואמר אל יתהלל חוגר כמפתח / ואולי כי למסגרת חידתי לא תמצא מפתח / ואמר החכם אין לאדם להתאבל על פיד יום / ולא לשמח בעת ימצא פדיום / כי הכל בגזרת הנורא והאיום / ולא תדע מה ילד יום /

ואשא משלי ואמר:

אל תתעצב דוד אל פיד יום



אל תשמח יום תמצא פדיום



עם אל עולם הסכן ושלם



הוא הנורא הוא האיום



וביום מחר אל תתהלל



כי לא תדע מה ילד יום.

ויאמר האיש עד מתי חידתך בתוך חכך תמנענה / חודה חידתך ואשמענה /



ואען ואמר הנה החידה / שים פניך להגידה / אמר נא מי האשה אשר פרצה עליה פרץ / ונולדה בבריאת השמים והארץ/ וביום אשר נולדה בבית נואפים שקדה / ובכל אשר עגבה עליהם בגדה / ובסוד עירין וקדישין לא עמדה / והיא בעולת בעל / ותחשק בזב ובבליעל / ובחשקיה מעלה מעל / איננה פועלת דבר אך מתפעלת / תחשק בזרים ובעלה גועלת/ לא ידעה אלמון יחרידנה / רבו שנותיה וזקנה איננה / הנה זאת חקרנוה כן היא שמענה / כי למען הראותך הובאת הנה /



ויאמר האיש הזאת חשבת למשפט בעל הלשון / החידה אשר חדת הוא החמר הראשון /

וישא משלו ויאמר:

אמרו ומה אשה בעלת בעל



יום נולדו הרים הכי ילדה



יום נולדה בית נואפים שקדה



ובחשקיה מעלה היא מעל

היא חשקה בזב ובבליעל



בכל אשר היא עגבה בגדה



לא ידעה אלמון ולא עמדה



לראות בסוד עירין ואלי מעל

לא פעלה דבר ומתפעלת



רבו שנותיה ולא זקנה



תחשק בזרים בעלה געלת

אענה חידתכם צפנה



סוד כי היו לי נואפה אולת



תקרא והוא הסוד אשר טמנה.

ויהי ככלות האיש להגיד החידה / ויחרד השר חרדה / וירע האיש עלינו תרעה גדולה / נקהלה עלינו כל הקהלה / ותהי צרתנו כפולה / כי אמרו השכנים / לרגם אותנו באבנים / והגבירה אשר היתה בחלון נשקפה / כסופות בנגב חלפה / ועיניה כמטר יערפו / יזובו מים ונחלים ישטפו / ושם גפנה לשמה / ותאנתה לקצפה / ועמדה עלינו קצופה / ולא ראינוה שנה רצופה / ואנחנו קלון מכבוד שבענו / ודרך שלום לא ידענו / ושבנו אל ביתנו בבשת ובכלמה / כעם הנכלמים בנוסם מן המלחמה / ויהי כי באנו אל בית השר כל רואנו יחרד וישום / כי פני השר אינם כתמול שלשום / ובכל אשר יראה יגער וילחם / ויקומו כל בניו וכל בנותיו לנחמו וימאן להתנחם / ותתצב אחותו מרחוק / ופערה פיה לבלי חק / אולי מצרתו תגרע ותמחק / ויאמר השר אין עת לשחק / כי הימים עברו תורה חלפו חק / ויושם לפניו לאכל ויאמר לא אכל / כי צרות לבבי הרחיבו ושבתי לחצי הדאגות למאכל / ונלאיתי כלכל ולא אוכל /



וכראותי כי כל מעשינו אל השר לא לרצון / הלכתי בחמת רוחי ולקחתי גדי עזים מן הצאן / רך וטוב זכר תמים / ותעש אמו מטעמים / וכראותי כי חמתו כמעט ספתה / לקחתי בידי המטעמים והלחם אשר עשתה / ואמר יקום אבי שר התעודה / ויאכל כי חלף עת הסעודה / וישבע ויאמר לתורה ולתעודה / אם תקום תמר לא אברה מידה /

וישא משלו ויאמר:

לבי ירעד רגלי ימעד



איכה אסעד דוד שה בריה



חי אני לו תקום תמר



לא אברה מידה בריה.

וישאלני השר הראית מה עשתה הגבירה הנחמדה / כאשר הגיד בן נעות המרדות החידה / ואען ואמר / ראיתי והיו עיניה יורדות מים / כאלו נבקעו מעינות תהום רבה ונפתחו ארבות השמים /

ואשא משלי ואמר:

לא ידעה נפשי בשורי ענני



עיני צביה בבכי ידמעו



אם הם ארבות השחקים נפתחו



או מי תהום רבה אשר נבקעו.

ויבך השר על צרת הגבירה / ויאמר האל יצילנה מכל צרה / כי ידעתי כי נפשה דלפה מתוגה / ויתפלל על היונה ממעי הדאגה /

וישא משלו ויאמר:

אלהים בעזך נחה אל מחוזך



עניה שביה נהר צר בזאה



אכלה הממה זמן עד לכמה



תלואת תלאה טלאה תלאה



ועת היא רמוסה ברגליו תהלך



למלך לדמיון זנב הלטאה



ולא חלתה יום זמן רע הרגה



ושבה במותה בריאה בריאה



כבודה אשר נחלתה ושבה



תמורת רפאה רפאה רפאה .

ויהי כראותי כי השר לא יתנחם / וכל רואיו יחריד ובעיניו יקחם / ברחתי מבית השר בנפש נבהלה / אל המקום אשר היה שם אהלי בתחלה.
