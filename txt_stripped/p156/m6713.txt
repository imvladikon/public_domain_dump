


	
	מחברות עמנואל
	



מחברות עמנואל – מחברת שמונה-עשרה / עמנואל הרומי


אמר המחבר: בהיותי עם השר מעין המתגבר / מפרק הרי התבונה וסלעיה משבר / היינו פעם בדברי חכמות נדבר / וחילים נגבר / ופעם על ריב לא לנו נתעבר / נדבר על כל דור ודור משפחה ומשפחה כפי הרצון / פעם בדברים ערבים ופעם בדברי לצון / ונחשב גדולים ממנו לרועי צאן /



ויהי היום ויאמר אלי השר אמר נא בחסדך / איך היה הענין עם פלוני איש סודך / כאשר שבח זכירתו על זכירתך / ושמירתו על שמירתך / ובעבור אשר גאות עשן התאבך / ומטרה לחציו הציבך / נאחז בקרניו בסבך / ואתה השיבות לו כדרכי טובך / אמר נא איך היה המעשה / ולא תחמל עליו ולא תכסה /



ואען ואמר היה פלוני איש סודי / היותר זכרן אשר ראיתי מעודי / לא שכח לעולם דבר ראהו / או מפי אחר שמעהו / והיה מספר לאשתו ולאוהביו את כל אשר קרהו / והיה מתפאר בטוב זכירתו / ושש על נעם אמרתו / כי האיש היה מדבר צחות / ולו מבחר הספורים והשיחות / ואם היה שומע איש דורש / או בתורה קורא ומפרש / היה זוכר כל דבריהם / עקריהם וענפיהם / ופניהם וכנפיהם / התחלותיהם והנחותיהם וכונותיהם / באותיותיהם / ובמלותיהם / איש איש ללשונו בארצותם בגוייהם / גם היה המדבר נוצרי או ישראלי / ערי וארודי ואראלי / כמוהו ידבר והוא לא פנים קלקל


וחילים יגבר / סוף דבר כל איש לפניו שב נאלם / ומלדבר במקומו בוש ונכלם / ולא נראית זכירה כזכירתו מן העולם ועד העולם/ ויהי היום דבר רוחו אותו / לפאר במקהלות זכירתו / ולולי האריך לשון עלי הייתי מקים דברתו / אמנם רצה להגדיל זכירתו בהשפלת זכירתי / ולהעלות אמרתו על אמרתי / אז בחמלי על מעלתי/ אמרתי ללשוני עזריני ואתנשא / כי לא משנאי יחרפני ואשא/ אמנם הוא אנוש כערכי / ועת הראהו כי שקר נסכו ולא נסכי/ אז דברה לשוני בחכי / ונראיתי לכל כי לא מחשבותיו מחשבותי / ולא דרכיו דרכי / אז נשאר מזכירתו בוש ונכלם /וזכירתי עלתה במדרגה העליונה שבסלם / והיה לי שם עולם /


ויאמר השר רציתי לשמע בפרט איך היו מאמריו / ותשובותיך אל דבריו /


ואען ואמר ידע אדני נזר המבינים / כי יום אחד עליתי השערה אל הזקנים / ושם היו אנשי חסד אנשי אמונים / בחיק המעלה אמונים / ואז התחיל לספר מן הגדולות כל איש חכם / ויתנבא גם פלוני בתוכם / והתחיל לפאר זכירתו על כל הזכירות / ולשים אותה למעלה מגלגלי המאורות / ויאמר שמעו שמים והאזיני ארץ / כמו שזכירתי פרצה בגבול הזכירות פרץ / ועמנואל דודי בעיני לא יוחן / כי הוא בעונות שכחן / לא יזכר מה שאכל היום על השלחן / והנני נשבע בחיי מי אשר אאהב ואהבתיו / כי אין איש אשר פעם אחת בימי ראיתיו / אשר כל ימי חלדי לא הכרתיו / ובראותי אותו מרחוק בשמו קראתיו / וכל קורותיו אשר קראוני עמו הזכרתיו / ואדע שם כל עיר וכפר וכרך / עד דמשק ועד חדרך / ואזכר כל הקורות והחדושים / ושמות האנשים והנשים / והמלכות והשרות והפילגשים / ועמנואל יעמד כאיש משתאה / לא יזכר דבר מכל אשר יראה / יראו עיניו ולא יחקר / וישמעו אזניו ולא יזכר / ורעיונותיו יחגו וינועו כשכור / והתחיל לספר קורות מלכים / ושרים וקצינים ונסיכים / והאריך לדבר דברי הבלים והתלים / עד ששבנו לדבר מן החתולים / עד אשר קצנו משמע / כמעט שמרב הצער העינים החלו לדמע / וכל עוד שהיה מחזר לזכר הודו / ולהאריך בכבודו / אך בי ישוב יהפך ידו / וכשמעי דברים לא כנים / וקצת העם היו כמתאוננים / ואחר זכירתי מרננים / אמרתי עת להשליך אבנים / ועפר בעפר / כי פלוני ברית האחוה הפר/

ואמרה אתה האיש המתפאר בזכירה / ותתן אותה על ראשך עטרה / איך לא תעלה על לבבך מה שיש לי מן הזכירה / בחמשה חמשי תורה / ומיהושע ושופטים ושמואל / וממלכים וישעיה וירמיה ויחזקאל / ומהושע ויואל /

ומעמוס ועובדיה / ומיונה ומיכה ונחום וחבקוק וצפניה / ומחגי ומלאכי וזכריה / ומדברי הימים ומדניאל ועזרא וחמש מגלות / ומאיוב ומשלי וספר תהלות / ואדע בכל מלה ומלה / הנקוד והדקדוק והפעלה והפשט והדרש והסוד / מקירות הספון ועד היסוד / וספרי הדקדוק כלם / למקטנם ועד גדולם / וספר השמים והעולם / וספרי אבן עזרא בדרכי החשבון / ופרוש קהלת ומאמר יקוו המים אל החכם רבי שמואל אבן תבון / וספר מורה הנבוכים / וספר תלמי בחכמת הכוכבים והמהלכים / וספר המדות והערכים ופרושי המשנה והחבור / וספר רבי אברהם הנשיא בסוד העבור / וחלק הדקדוק והשרשים לרבי דוד קמחי / האומרים לנפשי רני ושמחי / וספר צחות ומאזנים / הנחמדים להשכיל ותאוה הם לעינים / וספר החיוג' והרקמה / וספר ההשגה וההכלמה / וספר החזיון / וספר כלי הנחשת וההגיון / אתה שמת מגמת פניך בראותך קצין או שר / או מלך או נגיד או טפסר / לדעת מדותיו וטכסיסיו / וחיליו ואוכלוסיו / ותענוגיו ומשושיו / ואפוניו וגריסיו / ותארו ומעשיו / וקיניניו ונכסיו / וגבעותיו ורכסיו / ומאזניו ופלסיו / ופרדיו וסוסיו / וגביעיו וכוסיו / ותוריו וסיסיו / ומלבושיו וקרסיו / ויינו ועסיסיו / וברבורי אבוסיו / ואוצרותיו ומאבוסיו / ורצועותיו וכיסיו / ואכריו ואריסיו / ושהרוניו ושביסיו / וחרבותיו ותריסיו / וקרוביו ויחשיו / ושרי מסיו ומכסיו / וצדקותיו וחמסיו / ותפארת עכסיו / וצאנו וכבשיו / ועושי רצונו ומכעיסיו / ופילגשיו וסריסיו / ונשריו ופרסיו / ועזניותיו וכוסיו / ונציו ותחמסיו / וינשופיו וטוסיו / ורחמיו ופסיוניו / ותרניו ונסיו / ושריון קשקשיו / וחלוקו ומכנסיו / ופוליו ותורמוסיו / ועטיו וקולמוסיו / וקורותיו וכפיסיו / ומשקיותיו וכבוסיו/ וגלוייו וכמוסיו / ואבניו וחרשיו / וכסלחיו ופתרוסיו / ודיניו ונמוסיו / וחתוליו ורמשיו / ואני שמתי מגמת פני לדעת מן האנשים והנשים / טבע אברי הגוף וכח הנפשים / ומספר החושים / הפנימיים והמרגשים / והטבעים החזקים והחלשים / ומה בין הנמזגים והמתמששים / ובין הנגשים והנפגשים / וסבת הרפים והקשים/ ובתי הנפש והלחשים / ומן המדות הענפים והשרשים / ומן האברים השרים והשמשים / וסבת העזי פנים והבושים / והנבונים והטפשים / והנוחים והעקשים והפתאים וחכמי חרשים / לא אחוש לדעת צורת הפרשים / ולזכר תאר המלבושים / ואם הם בחורים או ישישים / ואם אכלו פולים או עדשים / ובשר סריס או תישים / ואם הם שרי עשרות או שרי חמשים / שרי הרכב או שלישים / עירומים או חמשים / עורים או חרשים / משלחים או פרושים / ואם להם כנים או פרעושים / ראה מה בין זכירתך לזכירתי / ושמירתך לשמירתי / ואמירתך לאמירתי / ויותר מאשר תספר מדבר הבאי בקרב מחניך / בפני שתי נשיך ובניך ובנותיך / ושגר אלפיך ועשתרות צאניך / אני אספר מדברי חכמה לפניך / עד אשר לשמע תצלנה אזניך /

ויותר מאשר תזכר מן השיחות הבטלות והחידות / אזכר מן החכמות והסודות / ומן החקים והעדות / וכמעט שלא תזכר בלבך שמות אנשים / כאשר אזכר משמות החכמים והמפרשים/ ולא תזכר כל כך ערים וכרכים וכפרים / כאשר אזכר אני מן הספרים / מלבד היות כל ספר כמה ענינים כולל / ובזה יתהלל המתהלל / ראה מה בין אמרך פלוני בנה זה היסוד / ובין אמרי פלוני פרש כך זה הסוד / ובין אמרך כך שם פלוני או אותו הכרך / ואמרי כך הקשה פלוני ופלוני כך פרך / ובין אמרך ממקום פלוני עד מקום פלוני כך וכך מילים / ואמרי על פה ספר תהלים / או אמרי מכוכב פלוני עד כוכב פלוני כך וכך שעורים מגבלים / ובים אמרך בשנת כך וכך לפרט היה שבע או רעב / לאמרי חמשים ואחד מיל שעור עלית העב / ובין אמרך כך וכך אכלנו אמש / לאמרי סבת קדרות הירח והשמש / אתה שמת מגמת פניך בלכתך בדרך לשאל / שמות הכרכים מן הימין ומן השמאל / ואני אדרך ברגלי הרחובות והשוקים / וההרים והגבעות והעמקים / ונשמתי תהלך אז על במתי השחקים / לא אחוש אז לדעת שמות הערים והכפרים / רק אשיב אל לבי החכמות אשר בספרים / ושם אמריך הספורים / לא אאריך בספור אכילת הברבורים / והנמרים החברבורים / ולא בסעדת הדגים / וביתר מיני התענוגים / אתה תזכר שמות הפועלים העצלים / ואשר אכלו בביתך גבינה ובצלים / ואני אזכר שמות החכמים / ודבריהם וחידותם על לוח לבי נרשמים / אתה זוכר צבאות הארץ והדגים אשר במים / ואני אזכר צבאות השמים / אתה זוכר איך אכלתם על הגל / ושכבתם במעגל/ ופלוני שתה מן היין וישכר ויתגל / ואני זוכר אלף ושנים ועשרים כוכבים הנתונים וארבעים ושמונה צורות שבגלגל / אתה זוכר צורת המלך וגדוליו / ורעיו וגואליו / וכנוריו ונבליו / וקצת מהיכליו / וחצרותיו וטירותיו / וחדרי משכיותיו / ואני אזכור הכהן הגדול מאחיו / ומעלותיו ושבחיו / ובגדי הקדש ומעיליו / וחשב אפדתו אשר עליו / ופעמוני זהב ורמונים על שוליו / והציץ אשר עליו / פתוחי חותם קדש לאדני / וההיכל לפנים ולפני / והאולם אשר לפני השער וקדש הקדשים / והבריח התיכון בתוך הקרשים / והדביר והכרובים / ומסגרות בין השלבים / וצלעות הבית והעבים / והארון והכפרת / ומזבח העולה ומזבח הקטרת / ושמן המשחה וקטרת הסמים / וחשן המשפט ואת האורים ואת התמים / וחלוני שקופים אטומים / מחזה אל מחזה שלש פעמים / והכפות והמחתות / והפתות והדלתות / והשלחנות והמנורות / וים הנחשת ועשרה כיורות / והעמודים והכותרות / והספות והמזרקות והמזמרות / והמלקחים והנרות / וההראל ארבע אמות / וחלונים שקופים אטומות / ולשכה ופתחה באילים השערים / ומחוצה לשער הפנימי לשכות שרים / ועשוי כרובים ותמורים / והלשכה אשר נגד הגזרה / ופני הכרוב לתמורה / ותמורים אל אילו / ושמונה מעלות מעליו / והאיש העומד בין ההדסים אשר במצלה / ומנורת זהב כלה / ועל ראשה גלה / ושני הזיתים מימינה ומשמאלה / והמגלה עפה / והאשה היושבת בתוך האיפה / והעטרות אשר לחלם ולטוביה / ולידעיה ולחן בן צפניה / והארבע מרכבות היוצאות מבין שני הרים / והסוסים ברדים אמצים לבנים ושחורים / ואם באתי לזכר כל הדברים / אשר על לבבי יעלו / יכלה הזמן והם לא יכלו / מלבד אשר אדע הכונות / והסודות הנכונות / אשר בכל הדברים הנזכרים צפונות / ולכן חדל לך / מזכר זכירתך באשר כבודך תחמד / ובמקום גדולים במעלה אל תעמד / לך וספר התליך לקטני אמנה / ולנשים המוזרות בלבנה / כי שכלך בדברים האלה קטן מהכיל / ואז תצליח את דרכיך ואז תשכיל /


ויהי כשמע פלוני סאוני סואן ברעש / חמד לעמד מצפון להר געש / והדברים אשר אמר לא אמרם / כי קבל נזק וצער ובשת בעבורם / ויאמר היום חשכו כוכבי נשפי / הן קלותי מה אשיבך ידי שמתי למו-פי / וקפץ פיו ונאלם / ועמד בוש ונכלם / ולו חרפת עולם / וכל העם קראו אחריו הידד / ובראותו כי שמשי זרחה נודד / ועמד בפנים נזעמים / ולא ראינוהו חדש ימים /



וישחק השר ויאמר מה מאד היטבת לדבר / אחרי היותו על ריב לא לו מתעבר / ככה יעשה לכל איש / אשר יתהלל וריח רעיו בעיני האנשים יבאיש /



ואמר אל השר ידע אדוני נזר הודי / כי לולי עניי ומרודי / ואשר חמלתי על כבודי / הייתי מכה אותו במטה אשר בידי / כאשר עלי האריך לשון / אמנם אוצרותי גרפם נחל קישון / והחסרון איננו מניח לי לישון / עד ישקיף וירא אדני משמים / וצרות לבבי ימאסו כמו מים / ולא תקום צרה פעמים / ותחת מחגרת שק מעיל הוד אעדה / והקול אשר אמר רדה יאמר רדה /

  

ואשא משלי ואמר:

  

אלי פדעני בעזך מידי



עני ותחת שק מעיל הוד אעדה



צו לזמן אל נאמן לי יעשה



שלום וקול אמר רדה יאמר רדה.

  

ויאמר השר מה מאד חטא הזמן / כאשר לא בנה לך בית נאמן / ויחלק כבודו לפני אליפז ותימן / וילבישם תכלת וארגמן / ולא יתן להם צרעת נעמן / וכאשר ישאלוהו בניו למי נמטיר המן / יאמר למהומן / ולא יתן מרדכי על בית המן / ואיך לא יחלק כבודו לאחימן / ולכלכל ולדרדע והימן / ומדוע לא חלק לך מן ההון לשבעה / כאשר חלק לך מזמרת הידיעה / ולבני הפילגשים נתן מתנות ולבני בריעה / ולאיש אשר יקרב אל כל בהמה לרבעה / ולא יתן להם כל ראש מקרח וכל זקן גרועה / ומן הקללות שבירמיה ארך היריעה / ואמר החכם ממומי זה העולם ומהבלו / שהוא לא יחן איש ואיש


כפי שכלו / רק יחן אנשים לבבם למרע / ומצדיק עיניו יגרע / יהרג יאשיהו וימלך חפרע / וימיר רע בטוב וטוב ברע /


וישא משלו ויאמר:

  

אמר החכם ממומי



זה העולם או מן הבלו



לא יחן איש מה שראוי



רק יוסיף לו או יגרע לו.

  

ואמר אל השר חנני אלהים ויש לי כל / כאשר עשה אותך על ההון יכול / ולו היה כל איש חכם כמוך וגבר / ובנדיבות ידו כידך תגבר / לא חסר איש בעודו דבר / אמנם הנדיבות לך למקנה / ידך תהיה בו בראשונה / ויד כל העם באחרונה / ובכל אשר התהלכתי מעיר אל עיר / לא מצאתי איש משנת הכילות יעור ויעיר / כי כלם יאמרו שלם ישלם המבעיר / ואמרה בעדם אלה בני שעיר / ואשר יקראו נדיביהם / וראשי אלופיהם / הפך לבם לשנוא / ולא מצאתי בם רק נוקם וקנוא / וכל אחד ואחד יעלים עינו / וכמו פתן חרש יאטם אזנו / אם אקראהו בדבר יועילהו קל חיש יענה / אם אקראהו בצרתה לי ערף יפנה / והזמן יוסיף כל אשר לו למשנה / ורעה אליו לא תאנה / ידמה לכלב איש חרש ברזל יישן לקול פטיש והלם פעם / ולא ירגיש שאגת אריה וקול רעם / וייקץ לקול השנים וינער כפעם בפעם / לא אדעה מה אחרית הזעם /

  

ואשא משלי ואמר:

  

דוד יום קראתיהו לעזרתו



גז חיש כצפור ממרום קנו



אך יום לעזרתי קראתיהו



החריש כפתן יאטם אזנו



נדמה לכלב איש חרש ברזל



בית הולמים שבתו ומשכנו



יישן לקול פטיש וירדם



ייקץ לעת אכל לקול שנו.

  

וידע אדני הדבר / כי מצאתי הנדיבים זולתך כיענים במדבר / ומנחתם ונסכיהם כמדבר / ולעולם


לא קבלתי בושה / רק בארץ מרשה / מנדיבים שלשה / מצאתים פתח השער / ורעיוני אחזו שער / כי הייתי רעב וצמא / סוער והומה / ואקרב אליהם ואמר היום נפלו לי בנעימים חבלים / כי הקרה לפני נדיבים אל אלים / ועתה רחמו עלי האצילים / מחלימי החולים / וגואלי הגולים / כי אין חפץ בכסילים / כי הפיל לא יקימהו כי אם הפילים / והענק לא יקימהו כי אם הנפילים/ ועתה חוסה על נדיבי אשר התגלגל / עליו אופן הגלגל / נטרד בטרדותיו / ולבש חרדותיו / ואל תחמלו על איש ישנה דתיו / שיורד על פי מדותיו /


ואשא משלי ואמר:

  

חוסו על נדיב התגלגל



עליו גלגל עם טרדותיו



אך לא תחמלו על עשיר



שיורד על פי מדותיו.

  

ויען האחד ויאמר אמר נא עקש / עד מתי תהיה לנו למוקש / לנדיב לא גמלתו איך תבקש / מדוע חק המוסר פרעת / ומעיל הבשת מעל פניך קרעת / וקלון מכבוד שבעת / הלא ידעת אם לא שמעת / כי אדמתך אם לא תעבדנה לא תוסף כחה לך / והנה היא צועקת למולך/ אם לא תשקני אני לא אאכילך /

  

וישא משלו ויאמר:

  

לנדיב לא גמלתו אל תבקש



תקדם לו ואז ישוב לגמלך



זכר אדמתך לך אמרה קום



והשקני ואחר אאכילך.

  

ויען השני ויאמר מה לי ולך איש מהתלות / כי תבקש לך גדולות / היום רבו עבדים המתפרצים / אשר אמר הנדיבות לוחצים / הלא ידעת העדרים אשר עלי רובצים / והאם רובצת על האפרוחים / או על הביצים / והנוגשים אשר אצים ורצים / כי לי בנים שלשה וחמש בנות גדולות / ומשאת החשק החלה לעלות / ומיראתי פן אחת מהם תמצא עלילות / ללכת אל גבעת הערלות / ובאות נפשה תמשך יד החשקים ותנאף / קניתי הטבעות ונזמי האף / והנטיפות והשרות והרעלות / וכסף אשקל כמהר הבתולות / ובטרח גדול אותם אפרנס / ונדחי צרכיהם אכנס / לולי כי מן השמים עושים לי נס / ואיך אהיה על בני אכזרי / ואשכח עצם מעצמי ובשר מבשרי / וישבעו זרים כחי ועצבי בבית נכרי / ואמר החכם האיש אשר יתן הונו לזרים / יחשב מן האכזרים / כי הקרובים בקרובים נעזרים / ודומה במעשהו אל העין הרואה השחקים / אשר בתכלית הרחק רחוקים / ולא יראה הקרובים/ אשר סביבו הם יושבים /

  

וישא משלו ויאמר:

  

אדמה איש אשר יתן רכושו



אלי זרים וימנע מקרובו



אלי עין אשר יראה רחוקיו



ולא יראה אשר ישכן סביבו.

  

ויען השלישי ויאמר שמעתי אומרים / כי כתוב באחד מן הספרים / אשר בארבעה ועשרים / לא ידעתי אם בשיר השירים / או בקהלת או במשלי / מסמרטוטי כלמי וממהדורי מלי / חשבת להריק הכסף המושב בפי אמתחותינו / וההון אשר לכדנו ברשתנו / ולקחנו מיד הזמן בחרבנו ובקשתנו / בדברים ערבים / ובקראך אותנו אצילים ונדיבים / ויצאת אלינו לשטן / ולכן דע כי ידינו אחות לוטן / לא חנם שנה הזמן דתך / רק על פי מדותיך / ולכן נשים היאוש והמניעות פרי מדברותיך /

  

וישא משלו ויאמר:

  

אשר קוה להרבות את נדיבו-



תך ברצות לך את משאלותיו



קפץ ידך ושים את המניעות



והיאוש פרי מדברותיו.

  

כשמעי כסתה כלמה פני / ופלגי מים ירדו עיני / ואמרה בקול נעצב / מי האמין כי אתיצב / לפני החלד והעכבר והצב / אקרא אליהם ולא יענוני / ותחת אהבתי ישטנוני / והזמן הולך וסוער ואותי יעשה כלה / ורואה אני את פני נדיביו והנה אינם כאשר בתחלה / ואתנה את לבי לראות סבת אלה התלאות / ואם בסכלותי או ברשעי עלי באות / וכל אשר חקרתי ואחקר אמצא / בנפשי דעה ומועצה / ועל ימי התעלמותה סופדת / ועל חיל תאותיה מורדת ובוגדת /

  

ואשא משלי ואמר:

  

הה כי ילידי הזמן בי נחרו



אינם אלי נפשי כמו היו תמול



כי אחשבה יום ולילה אעמול



אוחיל לברכות אל והמה אחרו

  

איכה אקומו והמה סחרו



ארץ אשר לא אדעה אם השמאל



אלך לימין הולכות המה למול



ותמול עבדוני ופני שחרו

  

ואתנה לב אם לרע מעלי



צרות זמן עלי כהיום שקדים



ואחזה צדקי וישר פעלי

  

אבדן ימי שכל בכל יום סופדים



אריב במו שחרות ואשליך נעלי



על חיל בחורותי בשכלי מורדים.

  

ויהי כשמע האנשים את דברי / לא חרדו אל מאמרי / ולא נכנעו לבותם / הרעו מאבותם / ויאמרו לי אי הנכאב אשר רשע חרשת / ולכן מעטה העני לבשת / איך פני אבותיך הובשת / ומדוע אהליך נטשת / האם אכלת כל אשר ירשת /



ואען ואמר נטשתי אהלי כי כן צויתי / ונפשי ממרורים רויתי / וקצתי בזמן ובו נבאשתי / כי ממצא חפצי בו נואשתי / ולכן בבשת פנים נגשתי / אל המחנה הזה אשר פגשתי / ולא אשנא העניות / רק בעבור היותי נכנע אל הפתיות / וצעקתי ולא נעניתי / ואולי כי לטוב לי ענתי /

  

ואשא משלי ואמר:

  

השאלים איך אהלי נטשתי



לא תדעו כי כן זמן צוני



הוא ממרור עם לענה הרוני



עד ממצוא חפצי אמת נואשתי

  

קצתי במחמדיו ובו נבאשתי



מה אמרה עוד הזמן הלוני



ימים ימי עצב עדי השוני



אל מחנות האל אשר פגשתי

  

אשנא זמן תחת אשר קנאתי



כי אחזה נפשי אשר כורעת



אל הר פתיות עד אשר נעניתי

  

ידע זמן כי אהליו שנאתי



על כן לקיקיוני ימן תולעת



אולי ידידי כי לטוב עניתי.



ויהי כראות האנשים חסרוני חשבו מלתי לאל /



ויאמר אחד מהם נדעה מה בפיו ישאל /



ויאמרו רעיו מה לנו ולצרה אם תגאל גאל /



ויאמר האיש ולשונו כאש תהלך / שאל מה אתן לך /



ואמרה בנפש שפלה / אחר בקשת המחילה / ידיך השר למודות בחמלה / בכפל מלשוני בשאלה /

  

ואשא משלי ואמר:

  

נדיבי עם הטוב כי תאמרו לי



שאל מנו ונפיק לך בחמלה



ידיכם הם למודות בנתינה



בכפל מלשוני בשאלה.

  

ויאמר האיש באה אתי הבית וסעד /



ואלך אחריו אל ביתו ולבי בצרת רעבוני ירעד /



ויאמר האיש התאכל מן הירקות החשובים /



ואען ואמר אלו הירקות טובים אכלום הכלבים /

  

ואשא משלי ואמר:

  

שאלוני במה תסעד לבבך



התבחר בירקות החשובים



עניתים הן יהי טוב בירקות



כלל עקר אכלום הכלבים.

  

ויאמר האיש הנני נשבע באשר אל יש הוציאני מאין / כי בביתי לא תאכל בשר ולא תשתה יין /



אז שתקתי והעלמתי עין / ואמרתי בלבי זה נשאר מבני קין / ויושם לפני מן הירקות קערה / וגרגרי זיתים עשרה / וצפחת ממי המערה / במדה ובמשקל ובמשורה / ופתותי לחם שעורים / ויהיו פי וידי לאכל נמהרים / ויבאו בי המים המאררים למרים / והיו סביבי שלשה נערים/ יקחו מלפני המעט אשר בעדי הכינו / וישובו וכל המשפחה עלי ילינו / ויאמרו לקח זה הרעבתן את כל אשר לאבינו / עודם מדברים והנה אשתו כאש יער תבער / תהמה ותסער / ובבעלה תריב ותגער / ותאמר זה ההלך בעיני לא יוחן / כי יאכל כאלו יטחן / לא יקום לעולם מן השלחן /



ויאמר האיש חי אני אם לא יניח האכילה זה הסכל / לא יפרד ממני עד יקיא את אשר אכל /



ויהי כשמעי קמתי קל חיש ופת לחם בידי /



ויאמר האיש הנה בשמים עדי / כי אם לא תניח הלחם אשסעך כשסע הגדי /



ואשליך הלחם מידי / ואברח /


והאיש אחרי מר יצרח / והנערים יהדפוני / והכלבים ירדפוני / והצרות מכל עברי הקיפוני / והמים עד נפש אפפוני / ועד פתח שער העיר רדפוני / ולו השיגוני טרפוני / וראה פניך ופני איש עוד לא פללתי / ועתה ראה השר איך נפלתי /


ויאמר השר באמת היתה זאת חרפה עצומה וכלמה / וכמעט שאין אחריה נחמה / ועתה הלא עשית באנשים ההם נקמה /



ואען ואמר נפתולי אלהים נפתלתי / וקרנם בעפר עוללתי / והשבתי מטהרם ונזרם לארץ חללתי / ובעמים חרפתם הודעתי / הגדתי והודעתי והשמעתי / כי לא די היותם אותי בוזים / רק שהקהילו עלי אנשים מחברי חרוזים / ויתקבצו עליהם אנשים רקים ופוחזים / מן היהודים הפרזים / אשר לא עמדו אבותיהם על הר גריזים / התפארו עלי בשיריהם / והדבר אין בהם / ישלחו לי שירים מטלאים / סבכים וכסבאם סבואים / יספרו בהם כי להם בכל התעודות / בכפל ממני עשר ידות / והיו בהם אנשים יולדו על ברכי וגמולי חסדי / למדו ממני דרך השיר והם מעשה ידי / ולא השיגו במרוצתם אבק שירי / ואין אחד מהם יזנב הנחשלים אחרי / ובראותי כי יניפו עלי יד שלחם / בשאט בנפש בגאון רוחם / בנים גדלתי ורוממתי ונשאתי טרחם / וקרבתי אליהם בברחם / ובעת קדר עליהם היום הייתי שמשם וירחם / ולא יזכרו מי שם ציץ נזר השיר על מצחם / אמרתי חי אני אבאיש ואביע שמן רקחם / ותשוב לשונם מדבק מלקוחם / ועל כרחם אהיה מלכם ומשיחם / ואקום ואחבר עליהם שיר נמרץ / בפי חזון נפרץ / הודעתי בעמים עזם / וכי הם יתר הגזם / והניפותי עליהם שיר חרבי / כאשר היה עם לבבי / והנה השיר אשר עליהם חברתי / כאשר התעברתי / ולוחות בריתי אתם שברתי / והודעתי אותם כי עליהם גברתי / והעלתי עד לשמים שיאי ואותם בעמקי שאולה קברתי /

  

ואשא משלי ואמר:

  

אשמע דבת רבים רבים



עלי יניפו יד שלחם



בי ישיחו לי יפיחו



בשאט נפש בגאון רוחם



בנים גדלתי רוממתי



זה כמה נשאתי טרחם



וכנשר יעיר את קנו



כן רחפתי על אפרוחם



זה משכרתי חלף כי כן



קרבתי להם יום ברחם



וביום קדרו כוכבי שכלם



הייתי שמשם וירחם



ובעת תעו בנתיבות בין



שכלי למחוז חפצם הנחם



לא ידעתי מי יולדתם



אם הקאת או הרחם



לא יכירו לא יזכורו



חסדי עמם יום נס לחם



אין בם יציץ מי זה שם ציץ



נזר הקדש על מצחם



מי רוח שכל בם מסך



או מי נסך את יין רקחם



איכה בי יריבו וזמן



משח אותי להיות פחם



לי חלק גלות עליות



מן שמים יום יום אלחם



אך להם גלות תחתיות



ברכות שדים ורחם



וזמני יום שם חטתי



שורה אז הפיץ את קצחם



ובעת קטרתי את מורי



הבאיש הביע את ריחם



ובידי מטה האותות



איכה אחרד לברק רמחם



אם מימי חכמותם כים



לבי הוא נחש ובריחם



בנשוא גלי מי ימיהם



אז מטה שכלי השביחם



וביום הציבו חיליהם



מצב חיל שכלי הבריחם



עליתי אל עיר גבורם



אף הורדתי עז בריחם



אנכי השויתי כיום



את שריהם אל פרחחם



החרש המסגר בגר



בתושבם ובאזרחם



חניתי כדור על חילם



ידעתי נועם גם נוחם



היה כיום נצחי נצח



אך חיליהם אבד נצחם



אזעק חמס אל הימים



ולחק גלגל כי השכיחם



מדוע דרכם צלחה



או מה ראה כי הרויחם



אלו הכיר מעבדיהם



או אלו ידע את שיחם



לא השכיבם על מטות שן



וסרוחים לא אסף נדחם



לא שעה אל מנחת הבלם



אף לא רצה כיום זבחם



לא נתן בר הלחי או



ממון נכנס בא על נכחם



על כן העלה על גורלם



ממון יוצא או הנלחם



על זאת לבי הוא יחיל בי



ובתוך קרבי יום יום יחם



כי הסכנתי כל ארחותם



זריתי רבעם גם ארחם



איך מול פריי פרח על שר-



שי הבין ישוו את קלחם



אם יראו חדקל שכלי



אז ימאסו מי שלחם



לו יריחו קטר בשמי



אז יבזו ריח ניחוחם



ישימו יד אל פה שרים



ולשונם מדבק מלקוחם



אקדיר שמשם אתן שרשם



כמק וכאבק דק פרחם



הנני לדור פרצם אגדר



אהיה להם ראש ומשיחם



ילכו גוים אל אור


שירי


ומלכים אל נגה זרחם



עשר ידות לי בחכמות



מי זה אתי יהיה נלחם



אקרא חם קר גם לקר אק-



רא חם ולהברד פחם



לי חכמה לי בינה אקרא



בה על כרחם בעל קר חם



לולי חסדי אז הפכתי



אל מערבם את מזרחם



איכה יבזו ערכי עם זו



ויאמרו כי ממי לקחם



אשאב שכל אבקת רוכל



אשיג נחש לי בריחם



איך תחת לחם אבירים



ודגן רום אקטף מלוחם



או תחת עדיי השכל



אחמד עדים או מלקוחם



איכה ירימו קרן או



עלי יניפו יד שלחם.

  

ויאמר השר באמת פרעת בם פרעות / ותחת אשר השקוך בשליש דמעות / שבעה נפשם ברעות / ובאמת היו כסילים / ובתכלית הנבלות נבלים / כי הנדיבים יתנדבו בגופים ובנשמות / ואף אם יהיו חסרי כל נעימות / לדמיון החרב החותכת העצמות / ואף אם תהיה בעלת פגימות /

  

וישא משלו ויאמר:

  

נדיבי עם הכי יתנדבו לך



ואף כי הם חסרי כל נעימות



כמו חרב בצאתה מתערה



אשר תחתך ויש בה רב פנימות.

  

ואען ואמר חייך השר היה בהם איש עשיר לרב / כי ירש נחלה מקרוב / והוא אמר ויבא ערב / והיה אוהבי בזמן שעבר / יותר מרחק נהר כבר /



ויאמר השר הלא שמעת משל הקדמוני / הכתוב בספר תחכמוני / שאמר די לך מעמיתך הנאמן / כאשר ינחילהו הזמן / ירשת עשר ומעלה רמה / אם יראה לך עשירית אהבתו הקדומה /

  

וישא משלו ויאמר:

  

יהי די לך ידידי מעמיתך



בירשו מעלה נשאת ורמה



ביום תתו והראותו לך את



עשירית אהבתו הקדומה.

  

ואען ואמר חי אני כי לולי חסדי אשר קדמום / לא חשבתי עליהם דבר מום / אמנם אני מעולם בהוני פדיתים / ובעת צרתם עניתים / והם לא די שלא היו בצרתי קרובים / אלא שדברו עלי כזבים / ולכן צר לי על החסדים אשר גמלתים / כי נחמתי כי עשיתים /

  

ואשא משלי ואמר:

  

בני נתן בני נבל בני שם



קדשתימו חשבתימו כזבים



אני תמיד פדיתימו ואפדם



והמה דברו עלי כזבים.

  

ודע השר אדני / ציץ מצחי ונזר זמני / כי כאשר הביאני / לחשק בתבונה / ולהיות ידי בחכמות אמונה / הוא ראותי היותי נמאס בעיני הסכלים / ונגש ונענה אל השפלים / וכל אשר יחשבוני לנבזה שוטני/ תגדל מעלתי בעיני / ובזה נודע לי חין ערכי / כי לו יחפצו הכסילים דרכי / אז יודע לי כי שקר נסכי /

  

ואשא משלי ואמר:

  

אשר גרם לנפשי לחשק את



תבונתי ורדפי טוב פעלים



ראותי כי אני נמאס לכל בז



ונגש נענה אל השפלים.

  

ואשר גרם אל הסכלים ההם / להיות מעלתי פחותה בעיניהם / הוא ראותם צרורות כספם באמתחותיהם / ויראו אמתחתי ריקה / בוקה ומבלקה / ונפשי שוקקה / ואמר החכם חסרון ההון יעלים הישר / וכן הסכלות יעלימהו העשר /

  

ואשא משלי ואמר:

  

אמר החכם יש חסרון



עשר יעלים רב הישר



כן יש סכלות עם הוללות



יעלים אותם רב העשר.

  

ושבחו הכסילים כי לא קנו העשר בשכלם / רק גלגל הוא שחוזר בעולם / ואם יחרץ חרוץ / ולקנות הון יחרד וירוץ / לא יצלח במעשהו / ודרך תבונות לא תועילהו / ויגיע ההון בנקלה ליד איש שובב / בשגעון ובעורון ובתמהון לבב /

  

ואשא משלי ואמר:

  

מה


תחרץ חרוץ ואראה הזמן


יאר פני חסדו לגר ולעקר



אם הגזרה יורדה ממעל



אם כן חריצות האנשים שקר.

  

ודמו החכמים בשכלם / העולם הזה לסלם / זה יורד בו וזה עולה / זה שוקט וזה גולה / זה נכבד וזה נקלה / ומקרא בצד ובכתף יהדפו / שנה על שנה חגים ינקפו / ומחסה כזב וסתר מים ישטפו / ואישיו למאמרו כסופה ירדפו / כלבוש יחליפם ויחלפו /

  

ואשא משלי ואמר:

  

ראה מקרי זמן בוגד



אנוש אנוש והפלא



ודבר ממך נעלם



אני את אזנך גולה



פרטיו לא אבאר כי



זמני מספר יכלה



זמן יריק לך מצוק



והטוב ממך יכלה



ואישיו חלפים יום יום



ואדם כלבוש יבלה



ומי יוריש ומי יעשיר



ומי ירד ומי יעלה



ומי מלך ומי הלך



ומי נכבד ומי נקלה



ומי עליון ומי אביון



ומי שוקט ומי גולה



ומי נטרף ומי נשרף



ומי נסקל ומי נתלה



מתי מספר כתבתים לך



ועוד נוסף עלי אלה.

  

ואמר שלמה בחכמתו פעם ופעמים / לכל זמן ועת לכל חפץ תחת השמים / ולכן הירא את דבר אדני מעבדיו / יזכה ארחו ויישיר מעבדיו / ועל דבר אמת וענוה צדק ישקד / ומעשיו ידרש ויפקד / כי הזמן יתן פני הצאן אל עקד / ותחת כבודו יקד יקד / וטרף זרוע אף קדקד / כי עת ספוד ועת רקוד /

  

ואשא משלי ואמר:

  

ראה צבי זמן ועת



ועל דבר אמת שקד



זמן ועת לכל יצור



ועת ספוד ועת רקד.

  

ויהי ככלותי לדבר / ושיר עליצתי לחבר / ויאמר השר הודעתני איך רבת שבעה לה נפשך יגונים / ולשמע לעג השאננים / והבוז לגאיונים אשר לא נשאו למעלתך פנים / ועתה הודיעני אם מצאת נדיב בדור / אשר פרץ הזמן יגדר /



ואען ואמר פגשתי שש עושה צדק / איש נדיב מחזק הבדק / תמים בדורותיו / יערפו כמטר נדבות מטרותיו / והזיל עלי מטל נדבותיו / ואכלתי בביתו דגים חשובים / וגדיי עזים טובים / וחברתי עליהם שירים ערבים /



ויאמר השר אדני / איך היה הדבר בני /



ואען ואמר היה בעיר אנקונ"א אדם נכבד / ולהתנדב עליו לא יכבד / מלך לנדיבות נעבד / וחשבתי כי כל הנדיבות במותו יאבד / ויהי היום והוא מצאני באש היגונים בוער / ובכל מחמדי הזמן גוער / כי ים התלאות הולך וסוער /



ויאמר אלי האיש מה אתה רואה / כי עמדת היום משתאה /



ואען ואמר חייך אני מרעיד עמדתי / כי היום לא סעדתי / ובכיתי וספדתי / כי ראיתי דגים גדולים ולשבת בצלם חמדתי / ואם לא אכל מהם הן גועתי אבדתי /



ויאמר האיש בחיי אם תדר לחבר עליהם שירים עליזים / יהיו לך הדגים ושני גדיי עזים / אז אמרתי עת לעשות חרוזים / אז הלכתי עם האיש אל חצר גנת הביתן / והראני דג גדול ראוי לשאול ויהונתן / ובראותי אותו חשבתי היותו לויתן / ואמר אל לבי אחשב היותי היום חתן /



ואמר אל הנכבד אם תעשה לי מטעמים ערבים / אחבר היום שירים מצור השכל חצובים /



ויאמר האיש הכון והכן לך השירים / כי המטעמים יהיו חמודים ויקרים / ויהי כאשר תגיע עת הסעדה / ישבנו אני והנכבד שר התעודה /



ויאמר השר יביאו הטלאים הנעימים / והגישו המטעמים / ונחליץ בהם העצמים כי הטלה ראוי למרום התהלות / ולכן שם אותו האל ראשון למזלות /



ואען ואמר חייך אדני דלפה נפשי מתוגה / ואין רפואה כדג לדאגה / כי לשון נופל על הלשון / וסכלו האומרים טלה ראשון /

  

ואשא משלי ואמר:

  

שים דוד לעמת הדאג דג



כי הוא לבד רפאות אלי נוגים



מה בערו האמרים טלה



ראשון למזלות ולא דגים.

  

ויאמר האיש חייך כבר הסכימו חכמי התכונה / ובכל שבטי ישראל מודעת נאמנה / כי דגל טלה נוסע בראשונה / ודגל דגים באחרונה / והנני נשבע כי דג לא יבא לפנינו / עד אשר בחכמתך תראנו / מעלת דגים על טלה / ואם הסוד הזה תגלה / להנחיל אהבי יש ואצרותיהם אמלא /



ואען ואמר דע אדני כי אין לטלה יתרון / בעבור היותו למזלות ראשון ודגים אחרון / כי למעלות הדגים וכוכביו העדינות / הושם באחרונה מאסף לכל המחנות / מצרף אל זה היות הכוכבים ואם רבו כחול / הכוכבים אשר בשנים עשר מזלות יהללו שם אל במחול / כי סובבים סובבים הולכים המזלות / בימים ובלילות / לדמיון אנשים יחלו במחולות / ומדרך אנשי המחול להתגולל כמזלות סביב / ובמחול האחרון חביב /

  

ואשא משלי ואמר:

  

אל עולם שם המזלות



כדמות מחול סובב סביב



על כן הושם דגים אחרון



כי במחול אחרון חביב.

  

ויאמר האיש הטבת את אשר דברת / והנני מקים האמר אשר גזרת / ויצו להביא בראשונה הדגים / ונהיה אוכלים ושותים וחוגגים / ואחר הדגים הביאו הטלאים הצלויים מור עובר נוטפים / כאלו מן השושנים קטופים / או משפתות היעלות חטופים / ומתוקים מדבש ונפת צופים / והמרקחות וזנים היו על השלחן כאבנים / והשקות בכלי זהב וכלים מכלים שונים / ויין מלכות רב / מתוק לנפש ולחך יערב / ואכלנו ושתינו עד אשר לא הכרנו מזרח ממערב /



ואמר יש לי רב / ברוך אדני צורי המלמד ידי לקרב /



ויאמר האיש ארור האיש אשר לא יהיה היום שכור / עד אשר לא יבחין בין הלתך והכור / ויאמר בדבור אחד שמור וזכור / ולא יבחין בין פתח תקוה לעמק עכור / למען זה היום כל ימי חיינו נזכור /



ואמר אל האיש אם תפטר אותי היום מלומר עוד שיר / אמר ברוך אדני ואעשיר / כי אחרי היותי שכור / וחבר שיר עוד לא אזכור / ויאמר האיש חברת שני שירים ואיך משיר תהיה שותק / וכבר למדתנו והחוט המשלש לא במהרה ינתק / ולכן חבר שיר שלישי / ותשקלהו במאזניך / ואחרי זאת להשתכר שים פניך / עד אשר ראשך עליך יכבד /



ואקום ואעשה מצות הנכבד /

  

ואשא משלי ואמר:

  

מתקו לחכי טלאים ודגים



ערכו לפני ידי הצבאות



הם בזמן מן לטעמם והנני



אודה לקדוש אדני צבאות



טלה בשומו לראש כל גדודיו



ודגים מאסף לכל הצבאות.

  

ויהי כהשלימי השירה / הלכנו וישבנו בעלית המקרה / ויושם לפנינו שלחן מעשה צעצועים / וכוסות מלאים וגביעים / ואהיה שעשועים / ועשינו מן היין משפטים / וכאשר באתי לקום מצאתי בטני ערמת חטים / ולא יכלתי לקום כאלו הייתי אסור ברהטים /



ויואל האיש גם הוא לקום / ובראותו כי לא יוכל ולא יקימוהו כל היקום / אמר נתנני אדני בידי לא אוכל קום / כי ראשי הולך הנה והנה / והיין שלח אש בעצמותי וירדנה /



ואען ואמר הנני נשבע בחיי נחור / כי תאבתי לשתות מי שיחור / כי היין שלח ידו מן החור / ופרש רשת לרגלי השיבני אחור / ולא אכיר הלבן מן השחור /



וכראות אנשי הבית יד השכרות החזקה / עשו בבית קול דממה דקה / ותפל עלינו תרדמה וישנו שנה עמקה / עד אשר פנה השמש לערב דרך גבולו / והשכרות חלף הלך לו / ועמדתי ימים שלשה עם הנכבד שמחת גילי / וזה היה חלקי מכל עמלי /



ויאמר השר נדיב היה נכבד / וזבד טוב אותך זבד / הדמו הגבירות אליו בנדיבות /



ואען ואמר בחיי אספו עם הנדיבות / כל ערבות וכל חשיבות / כי עמדו לשרתני בפנים יפות / ובזרועות חשופות / כשפחות חרופות / ולא סרו מעשות הבריה ולבב לביבות / והמטות היו באטון מצרים חטובות / ובכו לפרודי פעם ופעמים / כאלו נפתחו ארבות השמים / אמרו עד מה השר תשבנו / הנה נדודך יקרע סגור לבנו /

  

ואשא משלי ואמר:

  

אז תחנות לבך ידידים ישאלו



ואענה אותם בקמרינו



שם תאות נפש ושם ארינו



מר אהלות עם כל בשמים נעלו

  

שמה אסירים מתמותה נגאלו



שם עיר אלהים באמת שרינו



בשתי לביבות מן אשר ברינו



עם יעלות בגאון אדני צהלו

  

נפשי פדות דמע ראיתיו על לחי



יעלות ודודים לי כחליל יהמו



האמרים עד מה צבי תשבנו

  

מה תהמי נפשי ומה תשתוחחי



פקדי לשלום הצביות נאמו



אוי לנדוד יקרע סגור לבנו.

  

ויאמר השר הלא חסרת דבר בבית הנכבד /



ואען ואמר בחיי חסרתי דבר אשר עלי מאד כבד / כי ראיתי בבית השר שומים / כאלו נגזרו מן האורים ומן התמים / ובצלים מהללים / יחפירו ביפים הגלגלים / וגלדיהם עבים/ והמן בתוכם נצפן / ומראיהם ומעשיהם כאשר יהיה האופן בתוך האופן / והחציר עב ויקר / כמטה אלישע ויפיו לא נחקר /



ואמר אל הנכבד ואל אדר היקר / הדג והבשר טפל והשום והבצל והחציר עקר / הביאו לי מטה החציר כי רצוני לעשות בו המופתים / ושמעתי אומרים כי הוא טוב למעשה חבתים /



ויאמר האיש חייך בביתי היום שום וחציר לא תאכל / כי ריחם נלאיתי כלכל ולא אוכל / וזה השלחן לפני אדני / ולא יקרב בו דבר של גנאי /



כשמעי הייתי אל היגונים למאכל / מלבד שהנכבד החזיק אותי סכל / ואמרה היום לקח ממני מחמד עיני / כי השום והבצל והחציר מבחר עדני / ועתה אפלה נא ביד אדני /

  

ואשא משלי ואמר:

  

אראה אני המן ואראה השליו



אך אמרים לי סור ואל תקרב הלום



כל מעדנים הם בעיני כחלום



פגע גאון לבי כהיום שולליו

  

כי הבצלים הם ללבי גואליו



השום והחציר והבצלים שלום



ביתי ועל ראשי עטרת יהלם



כי בם לבבי הוא יכלכל מחליו

  

השום צבי ארץ ופרח כל יבול



החן ילדהו וההוד רקמו



עיש ובניה ברית לו כורתים

  

גלדי בצלים הם כגלגלי זבול



אופן בתוך אופן והחציר כמו



מטה אלישע לעשות המופתים.

  

ויאמר השר היטבת לדבר / ושירים לחבר / ברוך אדני אלהי ישראל / אשר לא השבית לשירים ולמליצות גואל.
