


	
	מחברות עמנואל
	



מחברת עשרים ושלוש




אמר המחבר: בהיותי עם השר ירום הודו / חשבו גואליו ורעיו היותי איש סודו / אהבתים ואהבוני / ולרופא ממחה חשבוני / ואלי יבא כל איש אשר לו הנגע / ומקרה רע או פגע / והייתי מרפא אותם ברגע / ויהי היום ויבא אלי איש מגאלי השר ומזרע המלוכה / לבקש על מחלתו ארוכה / ויגד לי כל נגעי לבבו / ויפצר בי לשים עיני עליו ולהסיר עצבו / כי היה ראשו ורבו בצוקה / ואסטומכתו בוקה ומבוקה ומבלקה / ואמר אל האיש חייך מעשות רצונך לא אתרפה / הנני מעלה לך ארוכה ומרפא / ואקום קל חיש לרגל המלאכה אשר לפני / ועוררתי רעיוני / ושנסתי מתני / להכין צרי אל מחלתו / להיות נקי מאלתו / ואכין לו רפואה נמרצת / מזמרת הרפואות מקבצת / היתה בספרי אפוקרט רובצת / עשיתיה על יד איש פקח / רקח מרקחת מעשה רקח / ואתן לו הרפואה תוך הצלחת / ואבחר לו מן השעות המשבחת / בהיות הלבנה בעקרב במעלה צומחת / וחשבתי כי באותו המשקה תהיה רפואתו / והסרת מראתו / ותתם חלאתו / כי המשקה ההוא לפי דברי החכמים / היה בו לשלשלו עשרים פעמים / ואמר אל האיש התבודד אחרי שתותך זה המשקה / פנה מכל עסקיך עד אשר גופך ינקה / ולא תטריד רעיונך בדבר מהדברים / וסגור דלתי ביתך מכל עברים / כי הרוח יגרם לעשות תקיעה ותרועה ושלשה שברים / וזרזתיו מכל הצריך / ובזה אין צרך להאריך / והוא לא כן ידמה ולבבו לא כן יחשוב / כי אחרי שתותו הרפואה יצא יצוא ושוב / וישא בנפשו עצות / לחבר בלילה ההוא שירים ומליצות / ולא סר האיש המסכן / כל הלילה מעשות כן / עד אשר לזאת הסבה / כח הרפואה נכבה / ולא פעלה בו דבר תועלת / ושכח מאמר אנשי החכמה כי מחשבה פועלת / ואולת כסילים אולת / ותהי הרפואה ההיא כאלה נובלת / ויהי באשמרת הבקר הלכתי אל בית האיש / בן מחפיר ומבאיש / ואשאלהו על אדותיו / ואיך פעל בו המשקה וקורותיו / וישבע האיש בחיי אבותיו / ויאמר המשקה והדיאה-פיניון אשר חשבתו נכבד ויקר / לא שלשלני כלל כל עקר / כשמעי עמדתי משמים / ונפלו עלי אימים / ולבי על הרפואה רע יחרש / ולא היה לי פה להשיב ולא מצח להרים ראש / ובעמדי מרעיד ומשתאה / אמר אלי האיש הנכאה / חייך הרופא חברתי הלילה שיר מפאר ונחמד / אשר כל הבא אליו ועמד / והנני שמח עליו כעל כל הון / והוא לא ידע היותו מכה בשגעון ובעורון ובתמהון / ויוצא מחיקו גליון / כתוב עליו שיר חברו האביון / חשב היותו על כל השירים עליון / וכאשר התבוננתי עליו ראיתיו מלא חרפה וכדי בזיון / ואמר אל לבי אין לתמה אם שמח יונה על הקיקיון / ובודאי זה הוא מפעל הנקיון / כי מה שהיה לו להוציא דרך בית חרון תחתון / הוציא דרך בית חרון עליון / כי השיר אשר חשב היותו במאזני צדק מאזן / כדרך שיר כל חרזן / היו מזיו שגיאותיו מלאים מפיקים מזן אל זן / הכתיבה היתה רעה / והמליצה היתה נחלאה וצולעה / והמשקל היה מעקל / והוא לא-פנים קלקל / אסף השיר ההוא כל חרפה וגנות / ושגגות לא במסכנות / עד אשר היו עלי לטרח נלאיתי נשא / וישאלני האיש ויאמר ההטבתי לראות ההשכלתי עשה /



ואען ואמר חייך בעל המליצה שכחת שם המרקחת כי קראת לדיאה-פינדיון דיאה-פיניון / על דרך הצידונים שיקראו לחרמון שריון / ולא תאמר כי לא פעלה הרפואה / כי אדרבא פעלה פעלה נפלאה / ומי שמע כזאת ומי ראה / כי הוציאה מגופך כל טנוף וטמאה / וכל באשה וצחנה וחלאה / בלא צער ובלא מכאוב / כאלו קסמה לך באוב / ועשתה אתך נוראות / וגבורות ונפלאות / וזה לך האות / כי מה שהיה לה להוציא מלמטה הוציאה דרך הלשון / וכזה לא היה מימות אדם הראשון / שא נא עיניך וראה השיר אשר הקאתו / איך יעלה באשו ותעלה צחנתו / אין לתמה אם היה לך / עם החלאים מלחמה ערוכה / וכבד באסטמכא / ודקירת העינים / ומעד המתנים / ורפיון השנים / וצלצול האזנים / וכיוצא בהם / מן החלאים למיניהם / אמנם יש לתמה איך היית עד כה בחיים / אשריך שנשארו מעתה אבריך נקיים /



כשמע האיש בוש ונכלם / וקפץ פיו ונאלם / והיתה לו חרפת עולם / ונסתר בירכתי ביתו / ואני הייתי נחפז לברח מאימתו / פן יעיר עלי כל חמתו /



ובשובני בית השר וראיתיו מרחוק / אז ימלא פי שחוק / ויתמה השר על רב שחקי / אשר לא מחקי / ויאמר שמתנו על רב שחקך תמהים /



ואען ואמר צחוק עשה לי אלהים / וכראותי כי השר לדעת המעשה יכסוף/ ספרתי לו כל המארע מראש ועד סוף /



כשמע השר נמלא פיו שחוק / והקול נשמע למרחוק
/ וישבע השר בחיי ראשו / כי זה המעשה השביע בצחצחות נפשו / ויאמר אלי הנני נשבע בנפש אבותי / כי לא נעם לי ספור כזה כל ימותי / ולעולם לא עשית גבורות נפלאות / רק הפעם הזאת / אמנם קשה בעיני שהפסדת ההוצאות / אשר הוצאת במרקחת / אף על פי שלא הוציא הסכל דבר מתחת /



ואען ואמר חייך השר אם יגיע מסכלותו שלא לפרע / אחבר עליו שיר שלא ראית כמוהו לרע / ובראותו אותו יאמר עת לקרע / ורעה עליו זאת מכל הרעה אשר מצאתהו מימיו / כי אפרסם סכלותו לעיני בני עמיו / ויהיה למשל ולשנינה / ולא יחסר כל ימיו מגנה /



ואקום במצות השר משיחי / ואשלח מלאך אל האיש לשלח אלי שכר טרחי / ושכר המרקחת המשבחת / אשר פדעתהו מרדת שחת / וימאן האיש וישבע בחי נפשו / כי לא ישלח אלי מאומה מרכושו / אז נתתי לחטא חכי ולשאל באלה נפשו.


  


ואשא משלי ואמר:


  


קום השר קח לך גליון



וכתב בו שיר משיר ציון



שיר חברתי עת נתתי



אל יוסף משקה אזוביון



חשבתי כי יריק מעיו



יפדנו מיד כליון



אמרתי לו דוד החבא



וסגר דלת האפריון



השמר מצנים פחים



כי זה לך מפיד יום פדיון



הבקר אור שחרתיהו



לראות אם ירד לטמיון



ומצאתיהו ובידו שיר



מלא סיגים עם שגיון



שיר מלא חרפה וכלמה



אין שעור אל הבזיון



כאשר אין שעור לפאה



לבכורים לראיון



ולחרפת סכלותו קרא



בשמו משא גיא חזיון



יען חשבו כי על תבל



אין ערך אליו או דמיון



אמרתי שמחת על שיר



תחריד צורתו כל רעיון



כשמח יונה הנביא



עת שמח על הקיקיון



ואשאל האיש ואמר



מה פעל הדיאה פינדיון



ויאמר לי חייך לא



פעל דבר הדיאה פיניון



החליף את שם המרקחות



כקרא אל הר חרמון שריון



ואען אל תאמר כי לא



פעל דבר הנקיון



כי בלתו היית אח או



רע אל מחלון או כליון



שור מראתך גם חלאתך



שיר הקאת איש האביון



כי תחת צאתו מתחת



יצא מבית חרון עליון.




ויאמר השר חברת שיר על חרפת שירתו / הלא תחבר שיר על נבלותו /



ואען ואמר חייך השר לו ידעתי היותו נבל בטבע / הייתי מוסיף ליסר אותו כחטאתיו שבע / אמנם במה שחטא לי אין להביא ראיה / משום מעשה שהיה /



ויאמר השר דע אל נכון כי הוא אבי אבות הכילות / ולא השיג אדם אבקו בענין הנבלות /



ואען ואמר חייך השר עליו שיר אחבר / ולהגיד נבלותו חילים אגבר.




ואשא משלי ואמר:

הואיל לבי באר זלות



שר יוסף כי למאד גדל



החל לספר אך בראותו



כי אין מספר לספר חדל



יום כלי לכילות עשה



סכה הוא בנה לו מגדל



כצדיק כרשע השוה



לא נכר שוע לפני דל



למד לקפץ יד כי לא י-



פל מידו גרגר חרדל.




ויאמר השר חייך מעולם לא הודעתני מעשה שמחה כזה / ובטוב עשית נקמה מזה הנבזה /



ואען ואמר חייך לו שמעת מפי / את אשר קרה לי בימי חרפי / עם חשוקתי הנעימה / אשר היתה בניות ברמה / ואיך עזרתני חכמתי עמה / אז נפלאת הפלא ופלא על המארע / וידעת כי לדעתי בחור בטוב ומאוס ברע / הושיעה לי ימיני וסגור לבה הקשה קרע /



ויאמר השר הודיעני איך היה המעשה / ולא תעלים ממני דבר ולא תכסה /



ואען ואמר חייך השר חשקתי בימי חרפי אשה נעימה / יפה כלבנה ברה כחמה / חשקתיה חשק עצום ואכזר / בחשבי להיות ממנה נעזר / והיא חשבתני לזר / אני רודף אהבתה והיא תברח / אתחנן אליה בלב נשבר והיא מרה תצרח / אני שוכן במערב והיא במזרח / ואם אשכן אני בצפון היא בדרום / ואם אני בשאול תחתית היא תמריא במרום / ואם אני במרום היא תרד לבאר שחת / ולא מצאתי לעולם עמה נחת / ויהי היום השיב הזמן את הגזלה אשר גזל / והביאני המזל / לרעות בגנים / וללקט שושנים / ולראות הפרחה הגפן הנצו הרמונים / ובהיותי הולך בגן לרוח היום עמדתי משתאה / ואשא עיני ואראה / והנה היפה עיניה כיונים על אפיקי מים / זורחת כשמש בחצי השמים / מלאה הארץ מזהרה זהר / והיא כמעשה לבנת הספיר וכעצם השמים לטהר / אז אם עלז לבי אין לתמה / וברכתי לגבה מעל גבה / ואקום ואלך בלב עלז אל היפה / אשר כמו שחר נשקפה /



ואמרה אליה אחר בקשת מחילת השגגה / שלום עליך הנאוה והמענגה / אשר בשושני הידידת סוגה / אמרי נא עדינה גברת ממלכות עד מתי תהיה נפשי הנוגה / על אפיקי מי ידידותך כאיל ערוגה /ואור יפעתך אותי יום יום עשק / ואני עשיתי בעבורך מה שלא עשה לעולם על חשוקתו חשק / ולו הואלת הייתי עפר רגליך לוחך ונושק / אחלי גברתי בפעם הזאת גלי אלי מצפון לבך הטמון בחבך / ולא תליני מחשבות אונך עוד בקרבך / אם תחפצי בחשקי הנני / ואם תמאני / חשקי חשק עפרה אחרת לעולם לא יענני/



כראות היפה אותי קורא מן המצר / והיא היתה במשעול צר / אשר אין דרך לנטות ימין ושמאל / וזכרה אשר עשיתי בעבור אהבתה גם משלשום גם מתמול / ותאמר בלבה יש-לי על האיש הזה לחמל / ולא כגמול ידיו עליו אגמל / טוב לי להסיר מלבו אהבתי לצמיתות / וכתב לה ספר כריתות / ולא ירדף אותי בשוקים וברחובות ובכל עברים /
כאשר ירדף הקרא בהרים / אבקש מזה הסכל /אשר לעשות לעולם לא יוכל / אנסה אותו בחידות אולי תהיין ידי שכלו מהושיע כבדות / אם אשמיעהו חידתי במשלי / והוא לא יוכל להגידה אלי / ותכל תלונתו מעלי /



ותפן אלי ותשא את קולה ותאמר / הוי החשק הנמהר והמר / אשר אלי תתמרמר / ובשרך עלי סמר / הנני נשבעת בחיי הנורא והאיום / כי בשתים תתחתן בי היום / אני אחוד לך חידתי / אשר לכדתי במצודתי / ואם יגזר המזל ותעמד בסודה / ותוכל החידה להגידה / הנני נשבעת באל עילום / כי הנני נותנת לך את בריתי שלום / ולא אוסיף החשקים עוד לקוב / רק אשים למישור לבבי העקוב / ארכיב אפרים יחרוש יהודה ישדד לי יעקב / ואם לא תמצא חידתי בשלמות / לא תקוה ממני לעולם נעימות / וביום ראותך פני תמות / הנני בצל אילן התפוח הזה אתלונן / אשר פריו לתרופה ועלהו רענן / וכלו שלו ושאנן / בזאת תבחן / אם בעיני תמצא חן / אם תאמר לי דברים אמתים / כמספר התפוחים אשר בו נתתים / ושקרים והתלים כמספר העלים / אז מאויי לבך אשלים / ואם אין זאת עיני מראותך אעלים / ואתה תהיה כאחד מן הנבלים /



כשמעי ברכתי ברוך מנחם אבלים / המחזיר אבדה לבעלים / ואען ואמר שמעי צורה ההדורה / את מספר החידה ואת שברה / הנני מגיד לך כמספר התפוחים דברים אמתיים / לכל התפוחים עקצים בם תלויים / ושקרים כמספר העלים להגיד אהיה נחפז / כל עלה הוא מזהב ומפז.




ואשא משלי ואמר:




צבית חן בתפוח תעורר



חשוקה לחסות תחת צלליו



קרב לך בל קרב לי עד אמרך



בתפוחיו אמת אף שוא בעליו



ראי ידות כתפוח בחשבון



ואל עלה ועלה פז בשוליו



בניב קצר אגל סודך יחידה



תני נא לי למנה שוק ועליו .




ויהי כשמע הצורה הנעימה / את מספר החידה העלומה / היתה רוח אחרת עמה / ותאמר אלי בלב נשבר / מיי יצא הדבר / יקשתי לי ונלכדתי / מצאת החידה אשר חדתי / קנית אותי היום בשכלך / ואור יפעתי אשר על כל אור מלך / נתתו זורח למולך / והנה אהבתי לעמתך ואצלך / לעיני בני עמי נתתיה לך /



ואען ואמר גבירתי ומה יהיה הערבון / אשר תתני אלי לשבר רעבון /



ותען ותאמר אחרי אשר הודיע האל אותך את כל זאת אין כמוך חכם ונבון / ארורות תהיין הצביות /
ויוני הגאיות / המעלימות עין מהחכמים והפקחים / ותעגבנה לסתיר עצתם עם העורים והפסחים / הנך דודי אשכול הכפר בעל השירים הצחים / ואני כשושנה בין החוחים / וכבר אמרו החכמים שפתים ישק משיב דברים נכחים /



ותמהר ותפשט את מעילה ותשלך תחת אחד השיחים / ותאמר אלי אתה תהיה על ביתי ועל פיך אשק / ולא תהיה מחשקי לעולם נעשק / ותקרב אלי ותשקני / שמאלה תחת לראשי וימינה תחבקני / ויהיו דדיה לי ערבים / ועד נטות היום התעלסנו באהבים / ותהי אהבתנו יום יום נוספת / הולכת ועודפת / עד אשר באחרית המות חטפה / ומאסף הנפשות החסידות אספה / עד פה הגיע מעשה היפה.



כשמע השר את אשר חכמתי עזרתני / ועל במתי היעלה הרכיבתני / ותניקני דבש מסלעה / ושמן מחלמיש צור לשבעה / אמר הנני נשבע בנוטה שמים כיריעה / כי אתה ראשון לכל סבל בית החשקים / ואין כמוך נוטה אהלי החכמה ויריעותיה מקים / אמנם הזקן הכביד עליך /ובסבתו אחרו פעמי מרכבות משליך / ומיום אשר חלית את רגליך / בין משוררי הזמן לא נשמע קולך / ורעש רכבך והמון גלגליך / כאשר שמענום בסוד אלוה עלי אהליך /



ואען ואמר אדני לו לדברי הקשבת / ובסוד נבונים ישבת / אז ידעת כי אין הדבר כאשר חשבת אתה / כי בענין השיר והמליצה ככחי אז וככחי עתה / ומתי חברתי שירים מתוקים / במענה השכל יצוקים / רק בהיות יצורי נאנחים ונאנקים / בחלותי את רגלי / ואז נשאתי משלי / והנם היום על כפים חקוקים / ועל גרגרות אנשי החכמה כענקים / וזהב השכל מעליהם מריקים /



ויאמר השר הזה חסדך את רעך / ומדוע לא הראיתני מעשה צעצועיך /

כי נסככם שקר ואתם אין



לי תשחקו עת כי יגוני צובאים

איך תחשו הזוללים הסובאים



הנני שכור חמה ולא מיין .

חרבו / מי זה נגד הזמן יחזיק מצב / מי יתיצב לפני מקריות אם לא יתיצב / היעמד לפני מחנהו /החלד והעכבר והצב למינהו / מה יעשו הצמאים אל מי המזמה / אם מענניו לא יזלו מטרות החכמה / ראינוהו ולא נמות / ומצאנו בחברתו על נעימות ושלמות / והיה מרב עשות חלב אכלנו חמאה / ראינו את המראה / ונאמר אין זה כי אם בכח הנבואה / יד אדני עשתה זאת וקדוש ישראל בראה / וכל אשר שתינו יספנו צמאה / ואדירינו העלו את שבחו על שפתם / ומחוקקינו במשענותם / ובסעיף סלע עיטם / כי דעת שפתיו ברור מללו / ונטעי אהבתו נטעו נוטעים וחללו / קצתם ילקו איש בלשונו / לרוות צמאונו / ויתר העם כרעו על ברכיהם לשתות / ומתדפקים על הדלתות / ויאמרו איש אל אחיו מן הוא ולחם אבירים / היורד מעל ראשי ההרים / מראש אמנה ממעונות אריות מהררי נמרים / שכחנו בו צרת זמננו ותהי לאין / נשאנוהו על כתף ושמרנוהו כאישון בת עין / למעלתו וליפעת אורו / כי כליל הוא בהדרו / מה נוסיף עוד לדבר חותם תכנית / וחשק אהבת מעלתכם על גבנו האריך מענית / ולתאר מעלתכם ידי העם כבדים / כי בכם החכמה והענוה והנשיאות רוכבים צמדים /



ושאלתך השר הנכבד מורנו רבי שמואל מאתנו / ליעץ אל הנכבד אחיך פלא זמננו / לשוב מהר אל מחוז חפצו / אל נוהו ואל ארצו / אף על פי שהשאלה היא לנו נגדית / למעלת הנזקין נשום להם בעידי עידית / לא נתן דמי לו / עד אשר יבא לחזות בנעם בני ביתו ולבקר בהיכלו / אף על פי שאין צרך כי תמיד נראהו מצר ודואג על עכבתו / ופלגי מים יורדות עיניו על גלותו / ולבו באש הנדד בוער / הומה וסוער / ומשתדל בכל עז להשלים מלאכת אדננו המלך שיחיה ואת אשר נגזר עליו כדי לשוב אל מחוז חפצו / ואל מעון רבצו / לרחף על גוזליו ולהפלגת חשקו לשוב אליהם ולהיות בביתו כבראשונה / יום אחד בשנה חשוב שנה / אמנם מלאכת אדננו המלך שיחיה אשר בידו ארכה מארץ מדה ורחבה מני ים / ומאמר המלך קים / ולהניף ידו אל מלאכת ההעתקה בעים / ולא ירפנה עד תמה / ולמלאת מצותו חובה עצומה / והאל המשפר שחקים בחכמה / התולה ארץ על בלימה / והנותן לעם עליה נשמה / יגדיל מעלתך ויעשה שלום לך / שלום יעשה לך / כנפש אדננו ונפש דורשך שלומך וטובתך כל הימים / בני קהל רומ"א החתומים / הנוטים אל מעלתך כנהר שלומים / וברכות כחול ימים / וכבוד כחפצך / מלא רחב ארצך / עמנואל.



כשמע השר המכתב הנבחר / אז קפץ ונשבע ולא אחר / ויאמר הנני נשבע באשר נתן טל ילדותך מרחם משחר / כי אתה בעל הלשון / ולשיר ולמליצה גואל וראשון / וכל משוררי הזמן נחל קישון גרפם נחל קדומים נחל קישון / ברוך אשר העמידך לנס עמים / ולהיות ניר לבניו כל הימים /


  


ואשב עם השר בשלוה ובמשמני המדינה / כאשר הייתי אתו באמנה.
