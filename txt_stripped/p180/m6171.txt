


	
	[מליצי אהבה]
	



[על פרידת אהבים]



È — — — / È — — — / È — — / / È — — — /È — — — / È — —


מליצי אהבה גרי לבבות / להורות אהבים דיני אהבות

בעת חברה ישושון אך בלבות / לקנאת הזמן הרע עצובות

חזם יתנודדו לזמן בטרם / יפלס לנדד כיום נתיבות

ויהמה ים-בכי לפני ידידים / ולהב התשוקה תוך קרבות

וזה יסער וזה יבער בלב סר / ויוביש נוזלי דמע בבבות

לך השר אשר אסף תבונות / וחכמות כאסף ביצים עזבות

לך חרדו לבבות מצלעות / ואנחותם לחלמיש צור מדיבות

ואם התפרדו היום גויות / הלא התלכדו המחשבות

ואיכה יחיו עוד אחרי זאת / פגרים נותרו מאין לבבות

ואיך בכו תמול שואת זמנים / בהטותם מתי-אהב לאיבות

והן נוסף נדד על-שנאיהם / ונתן משכנות חברה חרבות

ואמנם כל-ימי אהב עמלים / וקראו איש אהבות איש דאבות

ימי חברה באש-אהב משיקים / ולילות נד במר ונהי מעיבות

ולכן אפלה-נא כל-ימותי / ביד חברה לרעות או לטבות

והן זעף נדד אשא ואוחיל / לשוב ימים ולילות ממשובות

ויונף משמי חברת ידידים / אשר יעצרו גשם נדבות.
