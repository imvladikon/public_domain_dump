


	
	אופן
	



אופן / ר' אברהם אבן-עזרא


אל אל בני אלים / ינהרו ויקוו, // איש ממקומו לו / יקדו וישתחוו.

ברים כתרשישים, / תארם בלי שמצה,



יש כארי ראשים, / יש כשור ועוף נמצא,



יש על דמות אישים; / פעלם לאל נרצה –



יורדים וגם עולים / לעשות אשר צוו, // ללכת במעגלו / יכספו ויתאוו.

רצים בלי רגל / רק ינופפום רוחות,



כפם ככף עגל / נמשלה בתוך טוחות,



שם פועלם דגל / יתנו, ובמנוחות



דומות לגלגלים / יעתקו וגם ינוו, // חיים לעולם, לא / יגועו ולא ידוו.

מה נכבדו שירים / שוררו בני עליון,



יושבים במסתרים / מאנוש, ושם חביון



עז אל, שמו הרים / בם לנס בלי פדיון –



יפעל באין כלים, / מעשיו למי ישוו? // רק ספרו גדלו: / הוא אמת ולא התוו.
